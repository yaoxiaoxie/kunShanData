Section : CPU OC - Basic Function (60) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Note:OC function default On or Off depend on project requirement.Please contact PPA for confirmation.
Note:1.If project supports CPU OC, need to test the following test items. 2.Vantage:AMD project does not support CPU OC
1. Plug in AC.
2. Enter to OS, launch Lenovo Vantage UI / Legion Space / Legion Zone to check CPU OC option.
By default it is disabled on Balance mode. Make sure you can find CPU OC on option there. Note:OC function default On or Off depend on project requirement.
3. Switch Fn+Q mode to Performance mode.
CPU OC still keep default setting. Note:OC function default On or Off depend on project requirement.
4. Switch Fn+Q mode to Quite mode.
CPU OC on quite mode is not support.
5. Switch Fn+Q mode to Custom in Legion Zone/Vantage/Legion Space.
Verify CPU OC still keep default setting. Note:OC function default On or Off depend on project requirement
6. Plug out AC and switch Fn+Q mode.
CPU OC is not support on DC mode.
7. Plug in AC, Switch Fn+Q mode again.
CPU OC still keep default setting when in performance/balance mode. Note:OC function default On or Off depend on project requirement.
8. Restart OS and boot to BIOS Menu.
Check CPU OC(capability for X70/X80) is enabled on BIOS.
9. Set CPU OC(capability) to disabled,save and exit and then restart to OS.
10. Launch Lenovo Vantage UI / Legion Space /Legion Zone to check CPU OC option.
For X70/X80, verify CPU OC option disappeared in APP, and OC status is OFF. For X60, verify CPU OC is switched to OFF.
11. Switch Fn+Q mode to Performance, and then switch power plan to High Performance.
12.Stress CPU with AIDA64. (AIDA64 -> Tools -> System Stability Test -> check "Stress FPU" -> Start) Note:For models designed for reverse PPAB, TAT can be used if Aida64 loading is not large during double roasting.(TAT->Workloading->CPU->Power Level:100%->update workload->Start workload) Open Intel PTAT/AMD AVT tool, and set polling time to 50ms if needed, Log CPU frequency with iPTAT for 5 mintues.
Verify max non-oc frequency can be observed in the log file.
13.Swicth Fn+Q mode to Balance(and power plan to High performance), and log CPU frequency for 5 minutes.
Verify max non-oc frequency can be observed in the log file.
14.Swicth Fn+Q mode to Quiet, and log CPU frequency for 5 minutes.
Verify max non-oc frequency can be observed in the log file.
15.Enter BIOS menu and set CPU OC(capability for X70/X80) to on,boot to OS.
16. Launch Lenovo Vantage UI / Legion Space /Legion Zone to check CPU OC option.
For X70/X80, verify CPU OC option display in APP, and OC status is OFF. For X60, verify CPU OC is switched to On.
17. Switch Fn+Q mode to Performance,enable CPU OC(X70/X80) and then switch power plan to High Performance.
18.Stress CPU with AIDA64. (AIDA64 -> Tools -> System Stability Test -> check "Stress FPU" -> Start) Note:For models designed for reverse PPAB, TAT can be used if Aida64 loading is not large during double roasting.(TAT->Workloading->CPU->Power Level:100%->update workload->Start workload) Open Intel PTAT/AMD AVT tool, and set polling time to 50ms if needed, Log CPU frequency with iPTAT for 5 mintues.
Verify max OC frequency can be observerd.
19.Swicth Fn+Q mode to Balance(and power plan to High performance),enable CPU OC(X70/X80) and log CPU frequency for 5 minutes.
Verify max OC frequency can be observerd.
20.Swicth Fn+Q mode to Quiet(and power plan to High Performance), and log CPU frequency for 5 minutes.
Verify max non-OC frequency can be observerd.
 
Section : CPU OC - Advanced OC (20) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1.Press "FN+Q" switch to "Performance mode" and make sure CPU OC is ON. If APP is Legion Zone, please change Fn+Q mode to Custom.
2. Launch BIOS or LV/Legion Space/Legion Zone and set all CPU OC related settings to the max.
3. Save and exit.
Note: If the setting is not suitable, then the system crash or BSOD may occur, and the setting will be resotre to default.
4.After enter OS,perform S3/S4/WB/CB for each 3 times.
5. Run AIDA64 FPU stress for 2hours.
6. Press power button 15s(depend on project EC design) to recovery
Make sure the OC value and Thermal mode can restore to default, and OC parameters can restore to default value.
7. Set all Voltage Offset related settings to the min and others to max, repeat step 2 to step 5.
