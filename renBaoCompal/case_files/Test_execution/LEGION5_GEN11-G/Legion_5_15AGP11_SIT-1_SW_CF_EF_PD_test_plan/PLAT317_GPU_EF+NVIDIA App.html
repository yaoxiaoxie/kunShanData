Section : Preparation (10) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1. Download the test image and flash the test BIOS.
2. Go to the Device Manager to check NV Graphic driver has no error.
3. Enable hibernate function --> Control Panel -->Hardware and Sound -->Power Options -->choose what the power button does --> change settings that are currently unavailable.

4. Please observe the screen carefully during the test. Ensure screen has no corruption,hang or BSODs
 
Section : UI Check (90) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1. Launch the NVIDIA app
Ensure NVIDIA app can launch successfully

2. The initial launch screen will show a Steup Guide, please click "Next" to finish the settings
3. Click on "Home"
The page include Library and Discover
3-1. User can add the game to Library
3-2. User can open/get the other nvidia apps(like NVIDIA Broadcast, NVIDIA Omniverse ...) from Discover
4. Click on "Drivers"
It will show the newest driver page with bullet points of “what’s new” and “what’s fixed”

4-1. Download and install the newest game ready drvier


Note: if game ready driver is the newest, please skip

Ensure the driver can be installled successfully.
5. Click on "Graphics"
It will show "Program Settings" and "Global Settings" tab
5-1. Click the "Program Settings" tab
User can configure in-game graphics settings and driver settings in this page
5-2. Click the "Global Settings" tab
It will list the driver settings like RTX Dynamic Vibrance,BatteryBoost…

5-3. Try to set different values for these settings

Ensure the driver settings can change successfully

5-4. Click the "Restore" button on right-top corner

Ensure the driver settings can restore to default value successfully

6. Click on "System"

It will show the Displays,Video, Performance and My Rig tab


Note: The Displays option only show in Discrete mode(eDP under dGPU) and not show in HG mode(eDP under IGPU)

6-1. Click the "Displays" tab (If not supported, skip 6-1,6-2 steps)

It will show "Global Settings" and "Display Settings" sections

6-2. Change the settings like resolution, refresh rate, Display mode and G-SYNC if supported and apply.

Ensure the settings can be changed and applied successfully

6-3. Click the "Video" tab

It will show the RTX Video Enhancements settings with VSR and HDR options

6-4. Try to set the different values for VSR and HDR


Note: RTX HDR value can’t be modified if HDR is disabled or not support in windows display settings

Ensure the video settings can change successfully

6-5. Click the "Restore" button

Ensure the video settings can restore to default value successfully

6-6. Click the "Performance" tab

The page will display the "Statistics" for NV GPU and "Automatic Tuning"

6-7. Click the "My Rig" tab

The page will show the configuration information and NVIDIA Control Panel open link

6-8. Click the NVIDIA Control Panel open link

Ensure the NVIDIA Control Panel can open successfully

6-9. Click the "VIEW RIG DETAILS"

It list the system software information for the product and what features are supported

7. Click on "Redeem"


Note: please connect the network before click the item

Users can enter a coupon code to get rewards and Log in to check redemotion status
8. Click on "Settings"
It will show the Features, Notifications and About tab
8-1. Click the Features tab
The page include Overlay,Drivers, Games and Apps, Lanuage.

8-2. Try to enable/disable the toggles if available

Ensure the toggles can be enabled or disabled successfully
8-3. Change the lanuage settings to English or Chinese
Ensure the lanuage can change successfully
8-4. Click the "Notifications" tab
The Driver updates and Available rewards are checked.
8-5. Unchecked the "Driver updates" and "Available rewards"
Ensure "Driver updates" and "Available rewards" can be unchecked successfully
8-6. Click the "Manage Windows notification settings" link
It will display the windows notifications settings UI
8-7. Click the "About" tab
The page will show Early Access, Privacy and some links

8-8. Try to check/uncheck the settings if available and click the links

Ensure settings can be checked/unchecked successfully.
9. Minimize the NIVIDIA app
Ensure screen has no corruption,hang or BSODs
10. Click the icon in Taskbar to restore defaults
Ensure screen has no corruption,hang or BSODs
11. Maximize the NIVIDIA app
Ensure screen has no corruption,hang or BSODs
12. Click Maximize button again to restore defaults
Ensure screen has no corruption,hang or BSODs
13. Close the NIVIDIA app
Ensure screen has no corruption,hang or BSODs
14. Re-launch the NIVIDIA app
Ensure screen has no corruption,hang or BSODs
15. Put the system into sleep mode and resume
Ensure the NIVIDIA app still be opened
16.Try to click the Home, Drivers, Graphics, system, Redeem and Settings tab
Ensure screen has no corruption,hang or BSODs
 
Section : Gaming scenario (120) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1. Download and install a Geforce Game. (like LOL,PUBG and so on)
2. Launch the NVIDIA app-> Graphics, add the game program
Ensure the game can be added successfully
3. Launch the game and start a match
4. Go to the NVIDIA app to Optimize the game in Program Settings
Ensure the game can be optimized successfully
5. Continue to play the game for 10 min, then close the game
Ensure screen has no corruption,hang or BSODs
6. Open the nvidia app to change driver settings like RTX Dynamic Vibrance

7. Re-launch the game and start it for 30 mins

Ensure the driver settings can work well
8. Press the "Alt+Z" hotkey to swiftly access Shadowplay recording tools, Freestyle Game Filters, NVIDIA Highlights, photo mode, and performance monitoring tools.
Ensure screen has no corruption,hang or BSODs
