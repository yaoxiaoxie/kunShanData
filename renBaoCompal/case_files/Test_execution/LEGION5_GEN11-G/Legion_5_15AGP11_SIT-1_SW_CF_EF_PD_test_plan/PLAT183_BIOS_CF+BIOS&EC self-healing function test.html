Section : Preparation (5) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

Ensure ME should not be locked before run this case


Note:
1. Check with project BIOS team to provide ME tool.
2. Check ME lock/unlock status with MeInfoWin64.exe.


Check method:
a. Run MeInfoWin64.exe
b. Search tab "End Of Manufacturing"
c. Check ‘NVAR Configuration State’, ‘Flash Protection Mode’ status
d. If NVAR Configuration State is 'Locked' and Flash Protection Mode is 'Protected', the ME is locked.
e. If NVAR Configuration State is 'Unlocked' and Flash Protection Mode is 'Unprotected', the ME is Unlocked.
Ensure BIOS and EE design support self-healing function. Make sure BIOS already support crisis BIOS.

Cover system language Chinese and English

Get BiosSelfHealing tool and signed bootx64.EFI file from BIOS team. Then copy bootx64.EFI to U disk-> efi -> Boot folder to create EFI shell U disk. If you already have bootx64.EFI file in U disk, please replace with the signed file.


Note:If there is no signed bootx64.efi file, please disable secure boot to test.
Get special BIOS from BIOS team with destoried BIOS boot block and FVMAIN.

If the project has both single SSD and multi SSD configurations, need cover these two configurations.
