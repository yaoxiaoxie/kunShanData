Section : Preparation (10) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1. Update BIOS to latest formal version that support LOGO DIY.

2. Prepare all type and resolution pictures for test.
 
Section : POST time check (60) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

1. Set Logo DIY via vantage/PCM with "Show loading icon" option checked.
Note
1. If vantage does not support the logo DIY feature, skip it.
2. Please reboot your system after setting the logo DIY successfully.

2. Shutdown system and power on.

Make sure POST Logo displays correctly, no garbage, flicker, hang up occours.


Ensure OS loading circle should be the same with default logo behavior.
3. Get the POST time in task manager --> startup --> BIOS time.
4. Wait 5 minutes and repeat step 2-3, get 5 records. Calculate the average value.
5. Remove all pictures, restore POST logo to default.
6. Get the POST 5 times and calculate the average value.
POST Logo displays correctly.

7. Compare the average POST time.

Make sure the logo replaced POST time is no more than "the default POST time +1s".

8. Repeat steps 1-7 with "OS Loading Circle" option unchecked.

Make sure the logo replaced POST time is no more than "the default POST time +1s".


POST Logo displays correctlly, no garbage, flicker, hang up occours.


Ensure no OS loading circle shown during POST.
