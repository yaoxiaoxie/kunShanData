Section : 动态刷新率 (20) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Notes: 1.For "动态刷新率" need cover all panel vendor that support this feature and different refresh rates are supported. 2.When the refresh rate is adjusted, the screen is black and the mouse moves to the middle of the screen is normal
Click "动态刷新率" button and check the infos displayed on "动态刷新率" page.
"低刷新率", "自动刷新率" and "高刷新率" buttons will be displayed on this page and default values follow BIOS
Change the refresh rate by clicking the buttons.
The buttons of "动态刷新率" and the corresponding CG animation will be changed correctly.Make sure that the refresh rate mode corresponds to the System Settings.
Press hotkeys Fn+R to change the refresh rate.
Confirm the refresh rate mode is sync with OSD under smart engine.Make sure the OSD responds times <1s, only appear once and redundant operation are invalid in 3s
Notes：If support Fn+R 3.0，please test the following steps.
Go to System Settings--> Display-->Advanced display-->Choose a refresh rate, can check refresh rate status.
Change the refresh rate to"自动刷新率".
Ensure the unit in AC mode and scene not in"影音场景".
The refresh rate in system settings will be highest.
Enable the "自动探测场景未开启" button and make the scene detect to "影音场景" and play a 24FPS video.
The refresh rate in system settings will be the lowest.
Plug out the adapter.
Ensure the scene will detect to "影音场景". The refresh rate in system settings will be the lowest.
Close other apps and play DNF.
Ensure the scene will detect to "游戏场景" in 4min. The refresh rate in system settings will be 60Hz.
Change the refresh rate to"高刷新率".
The refresh rate in system settings will be the highest.
Change the refresh rate to"低刷新率" and close all software.
The refresh rate in system settings will be 60Hz.
 
Section : 色彩管理 (10) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

Click "色彩管理" button and check the infos displayed on "色彩管理" page.

For SE SDK is later than 1.0.130.06232:
X-rite shows:"出厂默认, 鲜艳, 网页办公, 摄影/数码影院, 原生数码影院, 摄影印刷, SDTV视频" buttons. And "出厂默认" is selected by default.
Tcon shows:" Adobe RGB, sRGB, P3" buttons. The actual displayed options and default value are by product design.


 


For SE SDK is older than 1.0.130.06232:
X-rite shows:"出厂默认, Adobe RGB, sRGB, P3" buttons. And "出厂默认" is selected by default in "通用场景".
Tcon shows:" Adobe RGB, sRGB, P3" buttons. The actual displayed options and default value are by product design.
If OS before 24H2, X-rite&Tcon will support "auto" mode.

Notes:If an option is missing, please check with the project which gamut is supported.

Switch between all of the color mode randomly.

Color mode can be changed successfully. Make sure the screen is responsive.



For X-rite: Click OS Settings->System->Display->Color profile to check the profile is consistent with the mode set in PCM.

Open Settings->Display->Color profile. (Please confirm by project if units support "自动管理应用的颜色", if not,please skip)

It will has "自动管理应用的颜色" button.

Switch the "自动管理应用的颜色" button to ON.

For OS from 24H2:


The buttons in "色彩管理" will be gray and unclickable.  And it will promt to disable ACM to switch "色彩管理" modes. If click the jump link "ACM" in prompt, it will link to Settings->Display.



For OS before 24H2:


Ensure "色彩管理" button will synchronize with the settings status.
Disable the "自动管理应用的颜色" button.
Ensure the mode of "色彩管理" will back to previous settings. And check the color management configuration file in Control panel will select corresponding profile.
Switch between all of the color mode randomly.
Color mode can be changed and kept successfully. Make sure the screen is responsive.

Change the "显卡省电设置' mode to different mode,such as "集显模式"、"智能模式"...  then click the color switch button. (Please confirm by project if units support "显卡省电设置", if not,please skip)

Confirm the display shows no abnormality. Color mode can be changed and kept successfully.


For Tcon: Make sure the screen is responsive.


For X-rite: Click OS Settings->System->Display->Color profile to check the profile is consistent with the mode set in PCM.

If SE SDK version is older than 1.0.130.06232, please skip the following steps.

Enable the HDR button in Settings->Display->HDR.

The buttons in "色彩管理" will be gray and unclickable. And it will promt to disable HDR to switch "色彩管理" modes.

Disable the HDR button.

The mode of "色彩管理" will back to previous settings.  And the buttons in "色彩管理" can be switched successfully.

Enable "自动管理应用的颜色" and HDR button.

The buttons in "色彩管理" will be gray and unclickable. And it will promt to disable HDR to switch "色彩管理" modes.

Click the jump link to Display settings, disable the HDR button.

The buttons in "色彩管理" will be gray and unclickable. And it will promt to disable ACM to switch "色彩管理" modes.

Click the jump link to Display settings, disable the "自动管理应用的颜色" button.

The mode of "色彩管理" will back to previous settings.  

Switch between all of the color mode in PCM randomly.

Color mode can be changed and kept successfully. Make sure the screen is responsive.
For X-rite: Click OS Settings->System->Display->Color profile to check the profile is consistent with the mode set in PCM.

Change all the X-rite Color profile in OS Settings->Display or in Control panel-颜色管理(select the display device of unit).

For X-rite, the mode in PCM will be consistent with Color profile in OS settings.

Connect the external display. Change the color mode.

Color mode can be changed and kept successfully. Make sure the screen is responsive.



(Notes：When connect to the external display，X-rite color management can not take effect in duplicate mode）

Press Win+P to switch different display mode. Change the color mode.

Color mode can be changed and kept successfully. Make sure the screen is responsive.

Disconnect the external display. Change the color mode.

Color mode can be changed and kept successfully. Make sure the screen is responsive.
 
Section : OLED节能 (20) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Notes：Before test the following steps，please turn off the "自动唤醒" and "自动睡眠" buttons in"唤醒和睡眠" page.
Click "OLED节能" button and check the infos displayed on "OLED节能" page.
"系统工具栏变暗, 应用背景变暗,显示屏变暗" buttons are selected "从不" by default. (If the unit is not OLED screen, "OLED节能" will not display.)
Switch these three switch options to "总是打开" in turn.
They will take effect in turn.
For single-screen unit
OLED saving for single-screen unit, If not support, please skip the following section.
Change"系统工具栏变暗" to non-default mode and keep the unit idle.
System toolbar will be dimmed after corresponding time.
Move the cursor to the system toolbar.
System toolbar will return to original brightness.
Change"应用背景变暗" to a specific time and keep the unit idle.
Application background and system toolbar will be dimmed after corresponding time.
Move the cursor on the app or app background.
Application background(the area other than the foreground app) will return to original brightness.
Change"应用背景变暗" to "总是打开" and keep the unit idle.
Application background(the area other than the foreground app) will be dimmed quickly.
Click app background(the area other than the foreground app).
Application background(the area other than the foreground app) will return to original brightness.
Move the cursor to the system toolbar.
System toolbar will return to original brightness.
Change"显示屏变暗" to 1min,"应用背景变暗" to 30s and "系统工具栏变暗" to 30s. Then keep the unit idle.
System toolbar and application background will be dimmed after 30s.Then after 1min, the entire screen will be dimmed.
Move the cursor and touch the toolbar.
The entire screen will return to original brightness.
Change"显示屏变暗" ,"应用背景变暗" and "系统工具栏变暗" to 2min. And set the all options in Screen&Sleep timeout to 1min.
The display of screen will shutdown after 1min.
Change"显示屏变暗" to "从不", "应用背景变暗" to 30s and "系统工具栏变暗" to 30s.
System toolbar and application background will be dimmed after 30s.
Switch between these three switch options at will.
They will take effect according to the set time.
For dual-screen unit
OLED saving for dual-screen unit, If not support, please skip the following section.
Change"系统工具栏变暗" to non-default mode and keep the unit idle.
System toolbar will be dimmed after corresponding time.(the toolbar of screen B and screen C will be separate timing)
Move the cursor to the system toolbar of screen B.
System toolbar of screen B will return to original brightness.
Move the cursor to the system toolbar of screen C.
System toolbar of screen C will return to original brightness.
Change"应用背景变暗" to a specific time and keep the unit idle.
Application background (the area other than the foreground app) and system toolbar will be dimmed dimmed after corresponding time.
Move the cursor to or touch the screen on the app background.
Application background (the area other than the foreground app) will return to original brightness.
Change"应用背景变暗" to "总是打开" and keep the unit idle.
Application background will be dimmed quickly.
Click app background(the area other than the foreground app).
Application background (the area other than the foreground app) will return to original brightness.
Move the cursor to the system toolbar.
System toolbar will return to original brightness.(the toolbar of screen B and screen C will be separate timing)
Change"显示屏变暗" to 1min,"应用背景变暗" to 30s and "系统工具栏变暗" to 30s. Then keep the unit idle.
System toolbar and application background will be dimmed after 30s.Then after 1min, the entire screen will be dimmed.
Move the cursor and touch the toolbar.
The entire screen will return to original brightness.(the toolbar of screen B and screen C will be separate timing)
Change"显示屏变暗" ,"应用背景变暗" and "系统工具栏变暗" to 2min. And set the all options in Screen&Sleep timeout to 1min.
The display of screen will shutdown after 1min.
Change"显示屏变暗" to "从不", "应用背景变暗" to 30s and "系统工具栏变暗" to 30s.
System toolbar and application background will be dimmed after 30s.
Switch between these three switch options at will.
They will take effect according to the set time.
Enable Cross-screen Browsing in 'User center-->Dual-screen setting-->Windows Management'.
Open Edge as focus window and use five fingers to click screen.
It will display cross-screen.
Change the "显示屏变暗" "应用背景变暗" "系统工具栏变暗" to non-default .
Confirm the function normally when using cross-screen.
Press Win+F7, set the unit to duplicate mode.
It will change to duplicate mode sucessfully.
Change the "显示屏变暗" "应用背景变暗" "系统工具栏变暗" to non-default .
Confirm the function normally in duplicate mode.
Change unit to Laptop/portrait/Tend/Tablet/landscape mode and switch "显示屏变暗" ,"应用背景变暗" and "系统工具栏变暗" options.
They will take effect according to the set time.And the feature is working properly.
