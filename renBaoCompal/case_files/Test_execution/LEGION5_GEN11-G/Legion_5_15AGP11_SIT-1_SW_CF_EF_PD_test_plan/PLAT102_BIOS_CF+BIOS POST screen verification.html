Section : LCD Self Test (10) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

1. Shutdown system with AC.

Ensure system go to S5 status

2. User press “Fn” + “Left-Ctrl" and power button in same time to boot

Eesure run LCD self-test mode

3. The LCD will display white, black, red, green, and blue in sequence (no need exactly in this order), and then it will automatically shut down.


Notes: There is no regulation on the number of running cycles. The starting color is also not specified.

The test will run even when the CPU or main memory is not working properly, with no display. If no colors appear, the LCD is a failure candidate.

4. Press power button to boot

Ensure system boot normally

5. Put system to hibernate

Ensure system go to hibernate status

6. User press “Fn” + “Left-Ctrl" and power button in same time to boot

Ensure run LCD self-test mode

7. Run LCD self-test under DC mode in S4/S5 status


Note:


Some project EC may have no power under DC mode, the LCD self test cannot be trigger, you can confirm with the EC team for this.

Ensure can boot to LCD self test mode

Section : POST time check from ship mode (20) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Power on system and enter OS, charge battery capacity to 10%-60%. Boot system to BIOS setup, enable secure boot and load default, save and exit, and then boot to BIOS setup again.
Make sure "Restore factory key" is not needed when enabling secure boot. Make sure secure boot is enabled.
Run EC tool to enable shipmode in Windows, then remove AC and shutdown system, wait for more than 1 minute and press power button to power on.
Ensure system cannot power on without AC.
Plug in AC, wait for several seconds.
Make sure charge LED is on.
Power on system, run "GetBIOSPOSTTime.exe" in command line as administrator, record the POST time as shipmode POST time.
Ensure shipmode POST time should not be more than 8.5s and shipmode POST time <= 1.5 * normal POST time

Check Device Manager.

Confirm no yellow mark in Device Manager.
Repeat above steps for 5 times.
Ensure shipmode POST time meet spec.
