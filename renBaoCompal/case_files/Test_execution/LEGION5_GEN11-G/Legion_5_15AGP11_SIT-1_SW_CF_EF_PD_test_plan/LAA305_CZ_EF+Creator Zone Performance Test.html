Section : APP launch time (10) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

1. First launch time:


Refers to the time recorded for the app’s first launch after oobe.

2. Subsequent launches (2nd-4th):


Record launch times for the 2nd, 3rd, and 4th launches.

Ensure the average of the three times does not exceed 10 seconds.

3. Exclusions:


Do not include time spent navigating to the "All apps" list (only start timing upon clicking the icon itself).

Section : Idle image generation (20) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1.Open HWINFO to record a new hardware log and set the polling cycle to 1000ms.
2. Open Creator Zone to enter create interface.

3. Minimize the UI to idle for more than 25 minutes, and open the Task Manager to check the processes.

Ensure that CZ_ImgAIGC.exe is no longer in the Task Manager process list.


Note that you must not click on the UI during the idle process; otherwise, the process will be restarted.

4. Open the UI to input the given prompts (this input process should take more than 10 seconds) to generate images, and use a timer to record the time.

Image created successfully.
5. Repeat idle image generation 2 times, every image generation requires over 25 minutes of idle time.
During the test if you find any obvious lag, please also report issue to TDMS.

6. Select an 512*512 image, save it to the editing page using x2, x4 super resolution, and record the time.

PRC1.0: Not support.
ROW1.1: The version after 01.05.107.0801 not support.
PRC1.5 & ROW1.0: Please do test in the editing page.
PRC2.0 or above: Please do test by Lossless Zoom function in the AI Editing page.
 
Section : Continuous image generation (30) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1.Open HWINFO to record a new hardware log and set the polling cycle to 1000ms.
2. Open Creator Zone to enter the create interface, and run a preliminary image generation first.
The image generation this time will not be counted in the time.
3. Use given prompts to generate images and use a timer to record time.
This image generation is recorded as the time for the first image generation.
4. Repeat image generation 2 times (IMMEDIATELY, within one minute), then stop recording and save the videos and logs.
During the test if you find any obvious lag, please also report issue to TDMS.
5. Select an image, save it to the editing page using x2 super resolution, and record the time.
If the version does not support super resolution or lossless magnification, please skip this step.
6. Select an image, save it to the editing page using x4 super resolution, and record the time.
If the version does not support super resolution or lossless magnification, please skip this step.
Please repeat step 1 - 6 under DC + Intelligent/Balance Mode
