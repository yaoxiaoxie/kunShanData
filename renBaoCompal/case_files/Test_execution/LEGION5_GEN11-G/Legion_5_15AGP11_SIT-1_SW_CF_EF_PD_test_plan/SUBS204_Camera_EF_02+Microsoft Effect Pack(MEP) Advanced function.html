Enable all functions in Windows Studio Effects in OS Settings. Check preview image in the OS Settings.
Ensure the effects are displayed correctly. Ensure there are no abnormalities in the preview screen.

Ensure the effects are displayed correctly. Ensure there are no abnormalities in the preview screen.
Move your body back and forth (don't go out of the box), check the preview image.
Ensure the framing window will re-center around the face. The re-centering motion will be smooth.Ensure the size of face is suitable.

Ensure whole camera preview has blur effect.

Disable "Automatic framing""Eye contact" in Teams.

Ensure "Automatic framing""Eye contact" effects disappear

Note:If MEP is the V1 version, it will only support two modes of background effects

Ensure correct changes in functional effects. Ensure there are no abnormalities in the preview screen.

Open all MEP features(If there are multiple options under the function, just choose any one option)

Ensure the effectiveness of MEP functions is normal
Ensure video chatting can continue normally
Ensure there are no abnormalities in the system

Switch power mode, check the video chat again.(cover all power mode)
eg:Switching using hotkey Fn+Q

Ensure the effectiveness of MEP functions is normal
Ensure video chatting can continue normally
Ensure there are no abnormalities in the system
Close video telephone
 
Section : Power management (210) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Before testing, switch to "Use system settings" in the Windows camera app
Open OS Settings and Enable all functions in Windows Studio Effects.
Don't close OS Settings page.
Open Quick Settings and preview
Ensure all functions are functioning properly. Ensure there are no abnormal situations such as delays in the preview display.
Close OS Settings and Quick Settings.
Open windows camera to preview a people.
Ensure all functions are functioning properly. Ensure there are no abnormal situations such as delays in the preview display.
Close Windows camera app.
Put system to the sleep(Cover lid close&power option)&Hibernate&Cold boot&Warm boot.
Ensure system can go to corresponding status.

Resume the system and open Windows camera

Ensure all functions are functioning properly.
Ensure there are no abnormal situations such as delays in the preview display.
Ensure the switch status in the Windows camera app is correct.

Disable "Automatic framing""Eye contact""Background effects" in Windows camera app.

If MEP is V1:Ensure the preview is displayed properly and without any extra effect.
If MEP is V2:Ensure the preview screen only has the effect of Creative filters function.
Don't close Windows camera app.
Put system to the sleep&Hibernate
Ensure system can go to corresponding status.

Resume the system

If MEP is V1:Ensure the preview is displayed properly and without any extra effect.
If MEP is V2:Ensure the preview screen only has the effect of Creative filters function.The effect of Creative filters has not changed either.
Close Windows camera.
