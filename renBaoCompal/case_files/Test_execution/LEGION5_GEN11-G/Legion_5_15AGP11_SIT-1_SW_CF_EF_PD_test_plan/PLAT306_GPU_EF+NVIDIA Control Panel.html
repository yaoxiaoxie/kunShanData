Section : Preparation (10) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Prepare test unit(s), download the latest image, check driver list, and make sure the NVIDIA graphic driver for Optimus is installed correctly.
Ensure Hibernation function is enabled.

Notice: This case is for NVIDIA Graphics platform, please omit if your unit not support NVIDIA Graphics card.

Notes: If the tested production has no NVidia Graphic Driver for Win10 Cloud OS or NV Graphic UI can't be found on Win10 Cloud OS, please skip and remark.
