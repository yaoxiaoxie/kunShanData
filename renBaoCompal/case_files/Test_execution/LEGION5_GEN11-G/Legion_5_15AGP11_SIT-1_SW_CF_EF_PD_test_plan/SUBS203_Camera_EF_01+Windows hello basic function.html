Section : Basic function Check (135) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Enable Windows hello
Boot to OS.
Open All settings-->Accounts-->Sign-in options.
Add account password and then add pin.
Ensure pin can be set successfully.
Active windows hello function.
Register your face via IR camera.
When enrolling,should be able to see preview of the normal color image. Setup should complete within 30sec. Ensure it can register successfully.
UNLOCK WITH WINDOWS HELLO
Sign out or lock the screen then move the face outside of the camera's field of view.
It should see the camera led is on.
Change the sign-in option to "Face" if it is not the current mode.
Should see the message "Looking for you" on the screen.
Move the face in front of the camera.
Ensure it can login successfully.
Reboot the test unit then move the face in front of the camera.
Should see the message "Looking for you" on the screen. Ensure it can login successfully.

For above two steps,please cover below scene:typical indoor office environment with overhead florescent lighting(indoor office),executive office with overhead florescent lighting and floor to ceiling windows(office with window),Dim lighting with 40-60w lights(living room),Strong ambient light from windows(kitchen),Ambient light from the surronding area(outdoor shade),strong ambient light,possible glare from sun depending on time of day(ourtdoor sun)

Ensure it can login successfully.
Sign out and log in with windows hello while you walking.
Ensure it can login successfully.
Sign out and login with windows hello while in a crowded area,such as Coffee Shop, Restaurant whereas some peopleare passing-by on the background
Ensure it can login successfully.
After login and rotate the device 90/180/270 degrees.(Tablet or 2 in 1.If your test unit is normal notebook,please skip below setp)
Sign out and try to login in with windows hello again.
Should see the message "Looking for you" on the screen. Ensure it can login successfully.
RE-ENROLL WINDOWS HELLO
Open All settings-->Accounts-->Sign-in options-->windows hello-->Face-->Improve recognition
Follow the pop up windows to complete the enrollment.
The orientation of preview image is correct. The re-enroll should complete within 30 sec.
Abnormal log in

Sign out or lock the screen.

Move another people face in front of the camera.(This people doesn't register the face)

Ensure it can't log in the system.

Move portrait picture in front of the camera.(This people register the face)

Ensure it can't log in the system.
