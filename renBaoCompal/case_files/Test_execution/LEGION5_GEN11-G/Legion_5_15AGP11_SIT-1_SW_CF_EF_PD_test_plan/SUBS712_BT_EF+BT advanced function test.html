Section : Preparation (24) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

1. Download the latest preload(If preload is ready);if preload is not ready,please make sure related drivers are installed and latest(wifi driver,BT driver,audio driver...)

Ensure base on Lenovo requirement(preload version is correct, related drivers' version is correct)

2. Check the driver version in device manager

Confirm the driver version is latest
3. Flash the latest BIOS
Confirm the BIOS status is ON after flash BIOS(If support)

4. Prepare all BT devices as BT device list, all marked "V" devices of BT device sheet must do below related tests, if you can't get some must device at all, please choose optional device to do related test instead(same version device)
5. If the test need cover more than one BT device,please repeat the related test one by one,not together
6. Test must cover all BT modules which project support
7. Please record the type of BT devices(contain aided units with all kinds of BT card) during below test,add note under related steps,this can make everyone review test result clearly

8. OS design: If the Bluetooth settings page is open, regardless of the "Allow Bluetooth devices to find this PC" option, the device is discoverable.

9. Make sure the Bluetooth settings page is closed after pairing so it doesn’t interfere with the file transfer rate and BT sound effect.

10. Sleep means S3/Modern Standby(MS)

11. If necessary when the unit/other PC is ready to receive a Bluetooth file, you need first to right click Bluetooth icon in the system tray->Receive a File

12. Connect 2.4GHz AP before doing BT test(Unless the test step has clear requirements for the network status)

13. All Bluetooth devices are connected via Bluetooth in the Bluetooth test(Unless the test steps has clear requirements for the connection method of the Bluetooth devices)

14. Please disable Windows update before testing:1.Windows key +R (RUN) -> ‘services.msc’ -> Windows Update -> Stop and Status type: Disabled 2.Task manager -> Services -> 'wuauserv':'Windows Update' ->Right click on stop

15. Before testing, check and record the version of the BT driver. After testing is completed, check and record again to ensure consistency between the two

16.Before run this test case , please run the 'CTA002_CF_09 BT test environment prechck' to verify the BT environment 

17.Please maintain consistency in the testing environment throughout the entire testing process

The BT mouse used smoothly and have no lag,the function of BT mouse is ok

BT mouse can be removed successfully
2. Bluetooth HeadSet/Speaker Connection

Note: For New Windows, Stereo and Hands free endpoints are combined in the UI, so omit step 2.34-2.35 and 2.37-2.39

2.1 Connect to a Bluetooth Headset(If project support BLE(BLE=Bluetooth Low Energy),Please cover classical BT headset and BLE headset,Both two kinds of headsets need to be tested)), if not support BLE, only test classical BT headset.

Bluetooth headset can be connected successfully

The default device should be the bluetooth device

2.3 Go to Control panel->Hardware and Sound->Sound->Playback devices,check bluetooth headset is the default device Note:If A2DP stereo headset,it can auto be the default device;but if HFP headset,need change it to be default device manually. If A2DP stereo headset, it is default speaker now; If A2DP stereo+HFP headset,it is default speaker now; If HFP headset,confirm can change it to be default device manually.

Sound can output from the BT headset smoothly

All function correctly

2.7 Select Bluetooth Device(default speaker)

Confirm can be recognied by system

Confirm this setting is correct

Confirm this repetition is successful

Power mode is "Best Performance(DC&AC mode)"

2.23 Connect to Bluetooth Headset

Bluetooth headset can be connected successfully,function is ok

2.24 Go to "Settings"->"Bluetooth &devices"->"Devices"->"Related settings"->"More devices and printer settings",Right click on the connected headphones and select properties,Go to the "Bluetooth" tab, fill in a different name after the headphones logo then click OK(Note:Need click "Disconnect" the headphones to modify the name successfully) then click "Connect"

Ensure that steps can be operated properly

2.25 Turn off unit Bluetooth then turn on

The headset can automatically connected and can be used smoothly

2.26 Remove connected Bluetooth headset 

The headset can be removed successfully

2.27 Put Bluetooth headset in pairing mode

2.28 Turn on unit Bluetooth in settings then connect Bluetooth headset

The headset can be connected successfully and the function of it is ok

2.30 Use phone to search and connect this Bluetooth headset

2.31 Use phone to rename connneted Bluetooth headset

The headsets' name can be renamed successfully

2.32 Turn off phone's Bluetooth then turn on unit's Bluetooth

The headset can automatically connected to unit and can be used smoothly

2.33 Restart test unit

The headset were also connected to unit and can be used smoothly

2.34 Change input device to headset hands-free audio on settings-System-sound-input

Ensure hands-free mode can be changed successfully

2.35 Change playmode from Stereo mode to Hands-Free mode by click speaker icon in task bar

Ensure hands-free mode can be changed successfully

2.36 Remove Bluetooth headset

Bluetooth headset can be removed

2.37 Restart unit -> connect Bluetooth headset -> change Bluetooth headset mode to Hands-Free

Ensure hands-free mode can be changed successfully

2.38 Play a music use default music player, remove Bluetooth headset during play music

Ensure Bluetooth headset has sound output and can be removed

2.39 Connect to Bluetooth headset again, change Bluetooth headset mode to Hands-Free

Ensure hands-free mode can be changed successfully

2.40 Repeat 2.23-2.39 3 times
