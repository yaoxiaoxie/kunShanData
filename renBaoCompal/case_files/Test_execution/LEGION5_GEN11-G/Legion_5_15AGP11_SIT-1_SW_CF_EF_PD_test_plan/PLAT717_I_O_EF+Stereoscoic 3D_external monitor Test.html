Section : 2D to 3D Conversion test (60) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1.Connect the test system and 3D TV via HDMI 1.4 or higher version cable.
2.Make sure 3D TV display is set to Max Supported Resolution
3.Set the 3D TV to 3D Mode via remote.

4.Install the latest PlayerFab.

5.Drag some 2D clips (.mp4, .MOV and .AVI) into the PlayerFab.
6. Click 3D icon on lower right corner -->3D source Format --> Side by Side Format

7.Put on 3D glasses which are compatible with 3D source.
8.View each clip for at least 2 minutes
Be able to view a 3D stream
9.Play each clip in full screen and windowed mode
No corruption, flickering, hangs, error essages, or other anomalies

10.Close PlayerFab.

5.Go to https://www.iqiyi.com/dianying/3dmovie.html
