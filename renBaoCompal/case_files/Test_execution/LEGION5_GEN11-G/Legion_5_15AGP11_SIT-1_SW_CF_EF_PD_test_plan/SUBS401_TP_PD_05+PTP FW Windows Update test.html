Section : Test preparation and notes (20) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
This case need be executed in 1 week before SIT exit.
1.Preload the latest images and OOBE.
2.Test UUT cover 32bit and 64bit , EN and SC with Lenovo OS, need cover all PTP device.
3.Flash the latest BIOS

4.All the test should cover both AC/DC, connect available WIFI.

5.Touchpad report rate can be checked by Digiinfo tool,PTP report rate SPEC follow MS WHLK SPEC(125Hz-250Hz)

6.Touchpad offline update FW file and TP FW UEFI value need contact with Touchpad owner to provide

7.Please test Touchpad FW offline update and ensure result is pass before test Windows update section(RD and vendor will sumbit FW to Microsoft after Touchpad FW offline update function is ok)

8. If testing offline PTP FW update section and online PTP FW update section at the same time , please preload system and update BIOS(MFG) after PTP offline FW update test result is pass, and then do PTP online FW update test.
 
Section : PTP Offline FW Update (80) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

Run cmd as administrator and input"bcdedit /set testsigning on"

Ensure test mode is on

Flash the original PTP FW

Reboot OS and check the PTP FW version

Ensure the PTP fw version is the original

Run enable test uefi.reg 

Ensure the registry key import success


The registry value="Drivers" in ”HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\DriverFlighting\Partner\TargetRing“

Open Device manager->Firmware

Check firmware device UEFI value(Firmware device property:Detail->Device instance path->UEFI->value) and select the right TP FW device 

Ensure the selected firmware device's UEFI value is the same as the value that represent TP FW device 

Right click firmware device,select Update driver ->"Browse my computer for driver software"->Choose the FW update file->"Next"->"Close"

Ensure PTP FW can be installed successfully

Reboot OS and check the PTP FW version

Ensure the PTP fw version is the latest

Check touchpad device in device manager

Ensure touchpad related device has no yellow bang in device manager

Run disable test uefi.reg 

Ensure the registry key import success


The registry value="" in ”HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\DriverFlighting\Partner\TargetRing“

Move cursor by touchpad and check Touchpad basic function

Ensure touchpad function(such as move,tap,hold,drag,gesture...) is OK and the report rate show correct.

Section : PTP Basic Function Check (120) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Move the mouse cursor and ensure the course moved smoothly and no crazy.
Open Paint APP
Move the touchpad cursor to the center of screen and hold on 10 seconds.
Ensure point pixel value is stability on the left bottom

Run LCD.exe on UUT

Tap on 13 points on touch surface, each point 10 times,every time wait 5 seconds.

Color of LCD.exe will shift following each tap smoothly
Put slug with 23mm on touchpad and move it.
Ensure mouse pointer can't move immediately
2 finger scroll in Edge
2 finger pinch in Edge
2 finger click in desktop
Ensure can trigger right click function
Three-Finger Swipe Left, switch app
Three-Finger Swipe Right, switch app
Three-Finger Swipe Up, Task view
Three-Finger Swipe Down, go to desktop
Four-Finger Tap,Invoke Action Center

Add new desktop and Four-Finger Swipe

Left Swipe from the desktop1 to desktop2 Right Swipe from the desktop2 to desktop1

Enter system to Sleep and resume, check PTP function and multi-finger function

Respond must real time in 2 senconds, All function(such as move,tap,hold,drag,multi-finger...) is OK after resume . Ensure the report rate show correct.

Enter system to Hibernate and resume, check PTP function and multi-finger function

Respond must real time in 3 senconds, All function(such as move,tap,hold,drag,multi-finger...) is OK after resume . Ensure the report rate show correct.

Reboot OS, check PTP function and multi-finger function

Respond must real time in 4 senconds, All function(such as move,tap,hold,drag,multi-finger...) is OK after resume . Ensure the report rate show correct.

Shutdown OS and power on, check PTP function and multi-finger function

Respond must real time in 4 senconds, All function(such as move,tap,hold,drag,multi-finger...) is OK after resume . Ensure the report rate show correct.
