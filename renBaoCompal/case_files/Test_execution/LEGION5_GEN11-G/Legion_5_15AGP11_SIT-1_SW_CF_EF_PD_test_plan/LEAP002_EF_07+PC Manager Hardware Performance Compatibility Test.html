Click "性能切换" button under "硬件性能" in"更多功能", and check the infos displayed on "性能切换" page.

For general projects:There will be "节能模式, 智能模式, 野兽模式, 极客模式"buttons.
For Yoga pro projects:The name will be "静谧模式" instead of "节能模式", "超能模式" instead of " 野兽模式". 



For all: "智能模式" button is checked by default. The checkbox "允许电脑性能自动提升到野兽(超能)模式" and "允许电脑自动进入智能省电模式"(display if supported) are is unchecked by default.



Notes: "极客模式"only shows when project supports，and if SE SDK is later than 1.0.130.06232, it will change name from "极客模式" to "创作模式" for Yoga pro projects.

Change the status to different mode.

The mode will be changed successfully. "极客模式" only can be enabled in AC mode. The mode can be sync with the status in SettingcarePA Tool.
Press hotkey Fn+Q to change modes.
The mode will be changed correctly, but can not switch to"极客模式".

Enter the BIOS to check the power mode value, switch between different modes.

The modes on both sides are the same.
Change the mode to “极客模式” in AC mode.
The button will be changed successfully.

Plug out the adapter.

The prompt“插入电源后可用" will appear and unclickable. The power mode will switch to the "野兽模式". (If Dispatcher driver is from 3.2.XXXX, it will be back to the previous mode）

Plug in the adapter.

The power mode will switch to the "极客模式".

Change the mode to "节能模式" ,then change to "极客模式" and plug out the adapter.

The power mode will switch to the "野兽模式".

Plug in the adapter.

The power mode will switch to the "极客模式".

Change the mode to "节能模式" ,then change to "极客模式" and reboot the DUT.

The power mode will switch to the "野兽模式".

Change the mode to "极客模式" , make unit into sleep then resume.

The power mode will keep to the "极客模式".

Change the mode to "智能模式" ,then change to "极客模式" and press Fn+Q.

The power mode will switch to the "智能模式".
Press hotkey Fn+Q to change modes in AC mode.
The buttons will be changed correctly, but can not switch to"极客模式".

Ensure in AC mode and run resident-evil benchmark(make sure GPU Usage >90%). Then use DPTF tool tool to check the power mode status. Downloadwebsite:https://store.steampowered.com/app/229950/Resident_Evil_6_Benchmark_ Tool/
The status will be in IEPM instead of APM.

Ensure in DC mode, and the battery is less than 30%. Then use DPTF tool tool to check the power mode status.
The status will be in IBSM instead of APM.

Section : 显卡超频 (20) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Notes: For "显卡超频""显卡省电设置"need cover all graphic card type.
Click "显卡超频" button and check the infos displayed on "显卡超频" page.
"显卡超频" button is OFF by default and the "高级设置" button is gray.
Enable the GPU OC in AC mode
The button can be turned on. The test unit's graphics type is displayed next to the "当前显卡类型"("当前显卡类型" will not be shown from the February 2024 preload version)
Click "高级设置” then check the overclock value for "显卡" and "显存".
"显卡" and "显存" manual overclock sliders will be displayed. Please check default and maximum overlock parameters is consistent with the project design.
Adjust the overclocking value arbitrarily, and use GPU.z to check the overclock value.
The tool displays the same overclocked value as "显卡超频"page.
Swipe both "显卡" and "显存" slider to the far right.
All of the displayed value are consistent with progress bar, the values will not be saved when OC is turned off.
Plug out the adapter.
The prompt "插入电源后，即可启用该功能。"will appear. And the button will be OFF and gray.

Plug in the adapter and change the status of "性能切换" to "节能模式".

For general projects: Prompt "手动切换性能模式至野兽模式，即可开启此功能"will be displayed on this page.


For Yoga pro projects: Prompt "手动切换性能模式至超能模式，即可开启此功能"will be displayed on this page.



For All projects:The button will be OFF and gray.The word "性能切换" will be blue and clickable .And it will link to "性能切换"page.
Change the status of "性能切换" to "智能模式".
The prompt "开启超频后，将自动切换至野兽模式。"will appear. Click “显卡超频”button to ON, the "性能切换" state will change to "野兽模式" automatically.
Click "显卡省电设置" button and change the mode to "集显模式".（This function is only displayed on specific project. If not support,please skip this step.)
The prompt "手动切换到智能模式或双显模式，即可开启该功能。"will appear. And the button will be OFF and unclickable.
Change "显卡省电设置" to "双显模式", open the task manager and end the wininit.exe in"详细信息", then click "结束进程树".
The unit will occur BSOD.
Click into "显卡超频“ page.
"显卡超频" button will change to OFF automatically after rebooting.
Change ITS mode to geek mode and enable the GPU OC.
Geek mode and GPU OC can be enable successfully.
Plug out the adapter.
GPU OC will auto disable. ITS mode will change to"野兽模式".
Plug in the adapter.
Geek mode and GPU OC will auto enable.
Change the GPU OC to off.
 
Section : 显卡省电设置 (20) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Click "显卡省电设置" button and check the infos displayed on "显卡省电设置" page.
The "显卡省电设置" mode is "双显模式" by default.
Open any game or creative software,such as LOL, resident evil, furmark,'剪映专业版' running on dGPU.
The NVIDIA graphics card model laptop GPU (非活动）checked in NV control panel can display the software running on the GPU.
Change the mode to "集显模式".
A pop-up prompts you to close the software running on the dGPU.Ensure can be switched to "集显模式" after all software running on dGPU is closed.
Change the mode to "智能模式" in AC mode.
The dGPU will load automatically.

Plug out the adapter.

If no software run on dGPU before switching to "智能模式"：the "显卡省电设置" status will be "集显模式".



If has software run on dGPU before switching to "智能模式"：the "显卡省电设置" status will be "双显模式".



Notes：Can check whether it is "双显模式"or "集显模式" by checking whether dual graphics cards exist in the device manager.
Change the mode to "双显模式".
Ensure can be switched to "双显模式"successfully, software can run on dGPU.
Connect to external display and refresh"显卡省电设置"page.
For project supports HDMI cable directly connected to dGPU chip, the"集显模式" will be gray and unclickable. If not support, the "集显模式" can be selected normal.
Disconnect the external display.
For project supports HDMI cable directly connected to dGPU chip, the mode will be kept "双显模式". If not support, the "集显模式" can be kept.
Make unit to NV DDS mode in NV control panel, check the"显卡省电设置"page.
The unit have no display exception.The mode will be "双显模式". "集显模式" will be gray and unclickable. Notes:Please check if support NV DDS mode (Open NV control panel,can find "管理显示模式->仅限NVIDIA GPU") , if not, skip the corresponding steps.

Make unit into sleep/Hibernate/warm boot/cold boot,then resume.

The display have no exception.The mode will be kept "双显模式"
Make unit not in NV DDS mode, check the"显卡省电设置"page.
Select "集显模式", then connect the external display.
For project supports HDMI cable directly connected to dGPU chip, "集显模式" will auto change to "双显模式" and prompt iGPU only mode can not be selected. If not support, the "集显模式" can be kept.
Disconnect the external display.
For project supports HDMI cable directly connected to dGPU chip, the mode will be kept "双显模式". If not support, the "集显模式" can be kept.

Make unit into sleep/Hibernate/warm boot/cold boot,then resume.

The unit have no display exception. The "显卡省电设置" mode can be kept. If select "集显模式" then do WB/CB，and the unit has self-starting process running on dGPU，the process will still be running on dGPU after reboot and app will prompt “集显模式” can not be switched.
