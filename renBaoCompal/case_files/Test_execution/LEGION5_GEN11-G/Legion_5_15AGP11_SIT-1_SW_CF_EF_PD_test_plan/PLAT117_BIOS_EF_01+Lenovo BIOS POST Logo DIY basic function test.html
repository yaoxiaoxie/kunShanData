Section : Logo DIY function check in Baiying ONE (60) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

Note:


1.Please confirm with project if system support LOGO DIY via Baiying ONE, if not, please skip below test.


2.This section will be supported for OTA 2.2.0 and later.

1. Open baiying ONE->Smart tool->Advanced toolkit



From Baiying3.0, Open Baiying ONE ->device management -> BIOS management

2. Click logo DIY function

Enter Power on screen customize page

3. Check maximum resolution

Ensure that the maximum resolution is the same as the system maximum resolution

4.Check the supported format.

Make sure supported format should be displayed.

5. Check option "show loading icon"

Ensure "show loading icon" is checked by defaults.

6. Add a picture image exceeds the system resolution or exceed the max size or format error

Make sure the image setup fails with an error message.

7.Manually modify the extension of the word or excel file to the supported image format, Logo DIY and select the image you just modified.

Make sure the logo DIY setup fails with an error message.

8. Click "CUSTOMIZE" button, select a picture which is supported format

The picture is displayed correctly.

9. View Screen Preview

Make sure the selected picture is displayed normally

10.Click "CLOSE" or "X"


From Baiying 3.0 not support "CLOSE" or "X" button, steps 10~11 can be skiped.

close logo DIY windows

11.Check DIY logo after restart/S4/S5

POST logo will restore to default logo, system show "Show loading icon" during POST.

12. Open logo DIY in baiying. Click "CUSTOMIZE" button, select a picture which is supported format, make sure that the resolution is no more than panel resolution, check "Picture Resolution" (each format please test 3 pictures with below steps).


From Baiying 3.0 Click "自定义" to select an image in the pop-up "insert picture".Then Click "open" to save the image.


 


Note:


1. Please cover project's all supported panel resolution configuration.


2. Please cover projects's all supported graphic configuration.

From Baiying 3.0, The logo picture displayed in screen preview and app will prompt to reboot.

13. View Screen Preview，click "CONFIRM"


From Baiying 3.0 not support 'CONFIRM' button, can skip it.

Make sure the selected picture is displayed normally, show notifiction: "Successful setup ! Please restart your device to view the LOGO"

14. Reboot system, check the POST screen.

Make sure Lenovo logo has been replaced by the picture that is stored in step 13.


Make sure DIY logo shows correct, no garbage,flicker, hang up occours.


Ensure OS loading icon show during POST.



Note: if some jpg format pictures cannot show normally after DIY POST logo, please check this picture EncodingProcess with online website: https://exif.tuchong.com/.(Refer attach picture). Please remark JPG format picture encoding process when open a defect.


For SMB products, after DIY POST logo, all POST remind message should hide, such as " To interrupt normal startup, press Enter" and etc...

15. Shutdown system and power on, check the POST screen.

Make sure Lenovo logo has been replaced by the picture that is stored in step 13.


Make sure DIY logo shows correct, no garbage,flicker, hang up occours.


Ensure OS loading icon behavior should be the same with default logo behavior.

16. Hibernate system and resume, check the POST screen.

Make sure Lenovo logo has been replaced by the picture that is stored in step 13.


Make sure DIY logo shows correct, no garbage,flicker, hang up occours.


Ensure OS loading icon should show during POST.

17. Press shift key and choose shutdown from start menu, then power on, check POST screen.

Make sure Lenovo logo has been replaced by the picture that is stored in step 13.


Make sure DIY logo shows correct, no garbage,flicker, hang up occours. Ensure OS loading icon should show during POST.

18. Open logo DIY in baiying. Click "RESET TO DEFAULT" button.

Make sure the picture displays the software's default picture

19. View Screen Preview

Make sure the default picture is displayed normally

20. Click "CONFIRM"


From Baiying 3.0 not support 'CONFIRM' button, can skip it.

Make sure the app prompts for a reboot.

21. Restart system and check DIY logo.

Make sure the picture displays the software default picture and is displayed normally.

22. Shutdown system and resume, check DIY logo.

Make sure the picture displays the software default picture and is displayed normally.

23. Hibernate system and resume, check DIY logo.

Make sure the picture displays the software default picture and is displayed normally.

Section : 联想电脑管家功能检查_Automation (35) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

PS: this section can execute via lenovo automation tool


Note1: If the link does not download test tool, you can get the automation tools from the case owner.
Note2: If you have any probelm during Automation tool running，can execute this item manually then inform case owner.
Note: Please confirm with project if system support LOGO DIY via PCM, if not, please skip below test.
打开联想电脑管家->工具箱->开机画面定制。
点击开机画面定制选项。
进入到开机画面定制界面。
检查最大分辨率。
确保最大分辨率与系统最大分辨率显示一致。
检查支持图片格式。

检查 “展示系统加载图标”



Note: For Home edition OS, "展示系统加载图标" will not show in the PCM which is except behavior

确保 默认勾选 “展示系统加载图标”
点击“自定义添加按钮”，选择超过最大支持分辨率的图片。
系统提示: *图片超出最大分辨率，请重新上传！
手动修改doc或excel文件后缀名为支持的图片格式，点击“自定义添加按钮”，选择刚刚修改的图片。
系统提示: *当前选择的文件不是图片，请重试!
选择支持的图片格式和分辨率的图片。
查看画面预览。
确保选择的图片显示正常。
点击取消按钮。
关闭开机画面定制窗口。
重启/关机/休眠，查看开机画面。
确保开机画面显示默认画面。

打开联想电脑管家->工具箱->开机画面定制。选择支持的图片格式和分辨率的图片。每种支持的图片格式各试3个不大于最大分辨率的图片。


注意


1. 请测试项目所有的屏幕支持不同最大分辨率的SKU。


2. 请测试项目所有的不同显卡配置的SKU。
查看画面预览，点击确定。
确保选择的图片显示正常。

重启，检查开机画面。

确保开机画面是刚刚选择保存的图片并且显示正常。


For SMB products, after DIY POST logo, all POST remind message should hide, such as " To interrupt normal startup, press Enter" and etc...
关机再开机，检查开机画面。
确保开机画面是刚刚选择保存的图片并且显示正常。
休眠再唤醒，检查开机画面。
确保开机画面是刚刚选择保存的图片并且显示正常。
按住shift键，选择关机（start menu），再开机，检查开机画面。
确保开机画面是刚刚选择保存的图片并且显示正常，
打开联想电脑管家->工具箱->开机画面定制。点击“恢复默认画面”选项。
查看画面预览。
确保开机画面显示软件默认的开机画面。

点击确定按钮。
重启机器，查看开机画面。
确保开机画面恢复默认开机画面并且显示正常。
关机再开机，查看开机画面。
确保开机画面恢复默认开机画面并且显示正常。
休眠再开机，查看开机画面。
确保开机画面恢复默认开机画面并且显示正常。
