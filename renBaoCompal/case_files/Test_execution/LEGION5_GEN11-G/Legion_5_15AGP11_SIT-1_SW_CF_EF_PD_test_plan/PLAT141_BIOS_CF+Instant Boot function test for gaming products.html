Section : Instant boot setting status sync check (5) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Boot to BIOS Setup
Press "Fn+F9" or click "Load Default Settings" to load BIOS to default setting
Switch to "configuration" tab, set the [Instant boot] to [Adapter]
Save and exit, boot into system

[方形电源口] setting is enable and [PD充电口]setting is Disable

set the [Instant boot] to Disable
Save and exit, boot into system

Launch Legion Zone app, check instant boot

[方形电源口] setting is Disable and [PD充电口] setting is Disable

set [方形电源口] to Enable, set [PD充电口] to Disable, exit Legion Zone
Restart the unit and boot to BIOS SETUP
Switch to the "configutation" tab, check the [Instant boot] setting
instant boot setting is [Adapter]
set the [Instant boot] to [Adapter and Type-C Power Delivery]
Save and exit, boot into system

Launch Legion Zone app, check instant boot

[方形电源口] setting is Enable and [PD充电口] setting is Enable

set the [方形电源口] to Disable and set the [PD充电口] to Disable, exit Legion Zone
Restart the unit and boot to BIOS SETUP
Switch to the "configutation" tab, check the [Instant boot] setting
instant boot setting is Disabled
Save and exit
