Section : Dynamic Boost 2.0 In AC (6000) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Notice: this section need cover the all Shipped adapter, two optional Slim adapter and two optional type-c adapter
When use the Maximum power AC adapter,please do single roast CPU/GPU and double roast test for 40 mins instead of 20 mins
GPU Single Roasting
1.1 Insert AC adapter to test unit.
1.2 Press "FN + Q" , switch thermal mode to Performance
1.3 Install the latest Furmark test tool
1.4 Launch NV GPUMon tool->PPAB State->Controller State, check it
Ensure the status is Enabled
1.5 Navigate to Dynamic Boost--->PPAB Enable, ensure it is enabled
1.6 Launch Furmark tool and click " GPU stress test " ----> "GO!"
if TGP can't meet the design spec, you can change some furmark settings to test
1.7 Navigate to GPUMon--->click Log Start -->Save as(Select the log path)--->Start

1.8 Keep running Furmark-Burn-in test for 20 mins, then close it and stop logging.



Note: Test duration can be adjusted according to the project spec to run a stable value

Ensure screen no BSOD,hang up, garbage…
1.9 Check the log file
1.Calculate the average TGP and fill in the attached table 2.Ensure TGP meet the DB on spec
1.10 Press "FN + Q" , switch thermal mode to Balance
1.11 Repeat steps 1.6-1.9
1.Calculate the average TGP and fill in the attached table 2.Ensure TGP meet the DB on spec
1.12 Press "FN + Q" , switch thermal mode to Quiet
1.13 Repeat steps 1.6-1.9
1.Calculate the average TGP and fill in the attached table 2.Ensure TGP meet the DB on spec
CPU Single Roasting

2.2 Press "FN + Q" , switch thermal mode to Performance

Ensure AIDA64 can run successfully

2.4 Intel CPU: Launch TAT to capture CPU power --> Monitor --> Monitor all --> Select "CPU Component" --> Start Logging.


AMD CPU: Run Systemdeck with administrator-->Unilog tab-->Start.    

Intel platform: "Package Power(Watts)" means the CPU TDP


AMD platform: "Socket power" means the CPU TDP

2.7 Press "FN + Q" , switch thermal mode to Balance
2.8 Repeat steps 2.3-2.6
1.Calculate the average TDP and fill in the attached table 2.Ensure TDP meet the DB On spec
2.9 Press "FN + Q" , switch thermal mode to Quiet
2.10 Repeat steps 2.3-2.6
1.Calculate the average TDP and fill in the attached table 2.Ensure TDP meet the DB On spec
