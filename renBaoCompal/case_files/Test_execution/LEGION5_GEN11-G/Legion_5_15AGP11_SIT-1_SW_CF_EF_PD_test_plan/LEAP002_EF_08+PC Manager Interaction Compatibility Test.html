Workloading : (Min) LeadingTime : (Min) Phase :
Notes：From SE SDK version is 1.0.110.12182, it will support "超闪硬盘". The feature is only displayed if unit memory size over or equal to 24GB, and it should has been installed the umcache driver. Ensure the umcache driver verison is the latest. The whitelist process please refer to the attachment.
Use unit with memory less than 24GB, has installed the umcache driver.
Confirm PCM will not display "AI智能场景-个性交互-超闪硬盘"
Use unit with memory over or equal to 24GB, has installed the umcache driver.
Confirm PCM will display "AI智能场景-个性交互-超闪硬盘"
Click "超闪硬盘" button in PCM and check the infos displayed on page.
The button is OFF by default. Notes:Please check the registry in Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\umcachedrv,if registry can find this value, the feature will be displayed.
Open the task manager, check system spare Memory situation. The threshold value of cache =70% Physical Memory Capacity- Physical Memory Used.
The cache of 超闪硬盘 will be choose 1GB,2GB,4GB based on the threshold.
Enable "超闪硬盘".
Ensure the cache threshold >4GB.
Check the Physical Memory Used will not increase 4GB in Task manager.
Run any whitelist process, and make the cache threshold keep >4GB.
The cache will be 4GB. Check the Physical Memory Used will be increase 4GB in Task manager.
Open some softwares to make the cache threshold keep < 1GB.
Check the Physical Memory Used will not increase in Task manager.
Make the cache threshold keep 2~4GB.
The cache will be 2GB. Check the Physical Memory Used will be increase 2GB in Task manager.
Make the cache threshold keep 1~2GB.
The cache will be 1GB. Check the Physical Memory Used will be increase 1GB in Task manager.
Disable the "超闪硬盘" button.
Check the Physical Memory Used will be released 1GB in Task manager.
User scenario

Ensure the function is in effect. Check Physical Memory Used will increase in Task manager.

Make unit into Warm boot/Cold boot, then resume.

Confirm the button can be kept. Check Physical Memory Used will be released in Task manager.
Run any whitelist process.
Ensure the function is in effect. Check Physical Memory Used will increase in Task manager.

Run any whitelist process.
Confirm the button can be kept. Ensure the function is in effect.The cache will be 1/2/4GB. Check Physical Memory Used will increase in Task manager.

Make unit into Hibernate, then resume.
Run any whitelist process.
Confirm the button can be kept. Ensure the function is in effect. Check Physical Memory Used will increase in Task manager.
Disable "超闪硬盘" and run any whitelist process.
Ensure the function is not in effect.Check Physical Memory Used will be released in Task manager.

Make unit into Warm boot/Cold boot, then resume. And run any whitelist process.

Confirm the button can be kept. Ensure the function is not in effect.Check Physical Memory Used will be released in Task manager.

Make unit into sleep/Hibernate, then resume. Run any whitelist process.

Confirm the button can be kept. Ensure the function is not in effect.Check Physical Memory Used will be released in Task manager.
Enable "超闪硬盘".
Ensure the function is in effect. Check Physical Memory Used will increase in Task manager.
