Section : Logging CPU, GPU Power (80) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Note: This section is a reference guide to logging CPU/GPU power on different platform.
Log Intel CPU Power
1. Install Intel Power And Thermal Analysis Tool (PTAT).
2. Run Intel PTAT as administrator.
3. Check "CPU Component".
4. Click on "Start Logging" to start to logging CPU power.
5. Click on the "Stop Logging" button to stop logging CPU power.
7. Check the log file in this path: This PC -> Documents -> iPTAT -> Log
The log is saved
Log AMD CPU/GPU Power
1. Install AMD System Deck.
2. Run AMD System Deck as administrator.
3. Switch to "PM" tab.
4. Click on "Start" button to start logging AMD CPU/GPU power.
5. Click on "Stop" button to stop logging AMD CPU/GPU power.
6. Check the log file in installation directory of AMD System Deck.
The log is saved
Log NVIDIA GPU Power
1. Copy the GPUMon tool to UUT.
2. Launch GPUMon by double click on the GPUMon.exe.
3. Click on the "Log Start" button to start logging GPU power. (Keep the checked item as default, and click on "Start")
4. Click on "Log Stop" button to stop logging GPU power.
5. Check the log file GPUMon.log.
The log is saved
 
Section : Single and Double Roasting (240) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Single Roasting_CPU
1. Start logging CPU power, SOP refer to the section "Logging CPU,GPU Power" (below the same)
2. Stress CPU with Prime95. (Refer to the attachment "Prime95 SOP New" for soecific settings)
How to run pls refer to attachment:Prime95 SOP
3. Stop Stress CPU with Prime95 after 20 minutes.
Ensure screen no BSOD,hang up, garbage.
4.Check the stable CPU power in the log, calculate the average CPU Power and fill it into the attached test reports.
1.Intel platform: "Package Power(Watts)" means the CPU TDP;AMD platform: "Socket power" means the CPU TDP. 2.Ensure TDP meet spec.
Single Roasting_GPU
1. Start to logging GPU power.
2.Stress GPU with Furmark2.0. Launch Furmark follow below settings: a.Graphics Test--->Furmark(GL); b.Resolution--->Select the DUT default resolution(if there is no DUT default resolution,select the closest resolution); c.Display OSI--->Check; d.More settings--->Uncheck"Back face culling"and"DXT5 texture compression(OpenGL)"; e.Anti-aliasing--->off Note:If the final result (Power/frequence/temperature) does not meet spec,Select "Furmark Knot(GL)" graphics test item,check"Back face culling"and"DXT5 texture compression(OpenGL)",set Anti-aliasing to 8X MSAA;
3. Stop stress GPU after 20 minutes.
Ensure screen no BSOD,hang up, garbage.
4. Stop logging GPU power.
5.Check the stable GPU power in the log, calculate the average GPU Power and fill it into the attached test reports.
Single roasting GPU power can meet SPEC.DC quiet mode,GPU power polling may has some tolerance,it is normal; tolerance is less than 5W can be accepted
Double Roasting_CPU+GPU
1. Start logging CPU and GPU power.
2. Stress CPU with Prime95. (Refer to the attachment "Prime95 SOP New" for soecific settings) Note:For models designed for reverse PPAB, TAT can be used if Prime95 loading is not large during double roasting.(TAT->Workloading->CPU->Power Level:100%->update workload->Start workload)
3.Stress GPU with Furmark2.0. Launch Furmark follow below settings: a.Graphics Test--->Furmark(GL); b.Resolution--->Select the DUT default resolution(if there is no DUT default resolution,select the closest resolution); c.Display OSI--->Check; d.More settings--->Uncheck"Back face culling"and"DXT5 texture compression(OpenGL)"; e.Anti-aliasing--->off Note:If the final result (Power/frequence/temperature) does not meet spec,Select "Furmark Knot(GL)" graphics test item,check"Back face culling"and"DXT5 texture compression(OpenGL)",set Anti-aliasing to 8X MSAA;
4. Stop stress with CPU & GPU after 20 minutes.
5. Stop logging with CPU & GPU.
There is no any problem under test, like hang upã€BSODã€device lostã€yellow bang and so on. Basic function are workable(Camera Keyboard,Video,Autio,HDD copy,Touchpad,Touchpanel,Sensors,Network,I/O port...)
6. Open the log file, calculate the average TDP+TGP and fill in the attached test reports.
Double Roasting: Ensure TDP+TGP can meet SPEC. DC:Ensure TDP+TGP meet the DBDC Setting SPEC.
