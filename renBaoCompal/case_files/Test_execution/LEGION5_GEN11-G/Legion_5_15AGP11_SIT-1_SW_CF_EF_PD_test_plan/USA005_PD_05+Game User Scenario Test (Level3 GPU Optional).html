Section : Preparation (180) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

Attention: This test requires both Chinese and English OS; Half the machines are set to Chinese system and half to English system.



All test are running under AC mode.

By default we suggest to run this test case under Performance (Gaming Projects) / EPM(Non- Gaming Projects）mode.



Some projects may have "超能模式" or Custom mode, please contact with your PPA to confirm which mode to do the test.
1. Refresh the latest BIOS.
Make sure the version is the latest in your test phase.

2. Preload OS, Please remove McAfee from Preload.

Make sure the version is the latest in your test phase.
3. Check out if the drivers are installed correctly.
Check device manager did not show yellow bang.

4. Download & install the test games.


* PUBG


* Apex Legends


* Baldur's Gate 3 


* Red Dead Redemption 2


* 黑神话:悟空


 

5. OS settings: Brightness 100%, maximum refresh rate，make sure battery capacity <90% and >60% when you start to do game test.


 

6. Please download the latest HWINFO (ZIP) version, Download link : https://www.hwinfo.com/download/



Please refer to HWINFO View FPS Frame Rate SOP.pdf to capture log. The log file MUST be in English.


Note: If HWINFO has compatibility issues on a new platform, you can use the following tools as backup options: Intel PTAT, AMD SystemDeck, or NVIDIA GPUmon.
7. Download"BatteryMonitor"Tool. Download link : https://pauto.lenovo.com/pa_tool_files/DEV/batteryMonitor/
Download the latest version
8. Refer to "BatteryMonitor SOP. pdf" to use "BatteryMonitor" tool
9. Download "Game Ladder" Tool. Download link : https://pauto.lenovo.com/pa_tool_files/performance/gameLadder/
Download the latest version

10. Unzip Game Ladder.zip to C:\

11. Please copy the latest HWiNFO64.exe to "Game Ladder\Packages".

12. After the test is completed, please fill in the test data in the Game test report. 


 
 
Section : P3 test(option) - Game Manual test (1080) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

Note:


1. Record the game information via "Game Ladder" tool, and feedback the log file with gaming report together.


2. Switch to Performance or EPM mode by pressing Fn+Q hotkey.


3. Set Win11 OS power settings option to "Best performance".


4. Use 'BatteryMonitor' to record battery information.


5. Make sure battery capacity < 90%,AC mode,brightness 100%,the highest panel refresh rate.


6. All game settings are set to "default settings".
Please confirm with the project team and PPA to conduct testing in "超能" mode or Performance mode
Please refer to the "Game Ladder" SOP to run the Game Ladder. The SOP is under \\Game Ladder\UserGuide.
PUBG
Make sure battery capacity < 90%,AC mode,brightness 100%,the highest panel refresh rate.
Run 'batteryMonitor. exe' as admin
Set FR=60/Cycle=120 for 'batteryMonitor'
Click on 'Start Log'
1. "PUBG" Testing.
1.1 Launch the game accelerator.
Make sure the game accelerator is working normally, and the network speed is stable.
1.2 Login the "Steam" and launch "PUBG".

1.3 Record the battery percentage when the game starts and when the game is over.

1.4 Use the Game Ladder tool to record the log by pressing hotkey "Ctrl + =" . If Game Ladder was failed to use, please log (Fraps+HWINFO) manually.

1.5 Play the game for more than 60 mins.

Make sure the game is running smoothly, with no lag and any other abnormal events.

1.6 Use the Game Ladder tool to end logging by pressing hotkey "Ctrl + =" .

Game Ladder log is created under \Game Ladder\Result and \Game Ladder\History

1.7 End BatteryMonitor Record

Save Log File

1.8 Close the game and fill the Game test report. Send the game test report and all logs files to wuhf2@lenovo.com and zhangjue4@lenovo.com to review.


 

1. AC: No battery capacity drop.


2. No BSOD, hangup,lag, screen flash or garbage.
