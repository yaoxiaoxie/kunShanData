Section : Driver Installation and Uninstallation (Win10 S OS skip) (20) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Please confirm the driver is for ''High Definition Audio'' or ''Sound Wire Audio'', select steps to test.
For High Definition Audio
1. Installation and Uninstallation notes.
Before uninstallation, go to Device Manager, select and unfold ''Sound, video and game controllers''.
Select codec's driver(eg.Realtek driver), Open Properties->Details->Extended Infs->find several oemxx.inf in the list, record them.
If support ISST, select ISST driver->Open Properties->Details->Extended Infs->find several oemxx.inf in the list, record them.
Please close all playback&record Apps
For step 2~3, after uninstall a driver or software component, DO NOT refresh to scan hardware changes, DO NOT restart system, for the driver will be installed back automatically and lead to the failure of uninstalling Extended Infs.
2. Open Device Manager, select and unfold ''Software components'', uninstall all software components that related to audio(such as Dolby, Nahimic, Fortemedia, Elevoc, Realtek).
All software components' drivers can be removed successfully.
3. Select and unfold ''Audio Processing Objects''(If existed), uninstall all APO that related to audio(such as Dolby, Nahimic, Fortemedia, Elevoc, Realtek).
All APO can be removed successfully.
4. Select and unfold ''Sound, video and game controllers'',uninstall codec's driver(eg.Realtek driver).
Audio Codec's driver can be removed successfully.
5. Open CMD or Powershell as administrator, input ''pnputil /delete-driver oemXX.inf /force'' command to delete all of these maping OEM inf files(they are recorded infs of Realtek driver before uninstallation) .
These inf files will show deleted successfully in CMD/Powershell.
6. Restart system,open Device Manager, select and unfold ''Sound, video and game controllers'', uninstall Intel SST driver, then select and unfold system devices, uninstall ''Detection Verification'' (If support), then delete the Extended Inf files of ISST driver with the method of Step4.4, reboot system, check ISST and Detection Verification driver in Device Manager. (If support ISST)
ISST and Detection Verification driver can be removed successfully.
7. Select and unfold system devices, uninstall Intel SST OED , then uninstall Intel SST BUS, reboot system, check these two drivers in Device Manager.(If support ISST)
These two drivers can be removed successfully.
8. Open audio driver folder on Disk C or D, open install file and complete driver installation. Notes: eg.For Realtek, if install file is a cmd file, please run it in CMD/Powershell(run as administrator).
No error in installation process.
9. Reboot system and check all drivers and software components&APO related to Audio.
All drivers and software components&APO that related to audio are list in Device Manager with correct driver version, and no yellow mark.
10. Open supported audio utility(such as Dolby, Nahimic...)
Audio utility can open without error.
11. Playback and recording function is OK
Overwrite Installation
1. Prepare a new version of audio driver, open install file(eg.cmd file) to overwrite installation of old driver
No error in installation process.
2. Reboot system and check all drivers and software components&APO related to Audio.
All drivers and software components&APO that related to audio are list in Device Manager with new &correct driver version, and no yellow mark.
3. Open supported audio utility(such as Dolby, Nahimic...)
Audio utility can open without error.
4. Playback and recording function is OK
For Sound Wire Audio
1. Please close all playback&recording Apps
2. Open audio driver folder on Disk C or D, open uninstall file and complete driver uninstallation. Notes: eg.uninstall file is a cmd file, please run it in CMD/Powershell(run as administrator).
All drivers and software components&APO that related to audio can be removed successfully(eg.Intel SST, Dolby, Elevoc, Cirrus)
3. Open install file and complete driver installation. Notes: eg.install file is a cmd file, please run it in CMD/Powershell(run as administrator).
All drivers and software components&APO that related to audio are list in Device Manager with correct driver version, and no yellow mark.
4. Open supported audio utility(such as Dolby, Nahimic...)
Audio utility can open without error.
5. Playback and recording function is OK
Overwrite Installation
1. Prepare a new version of audio driver, open install file(eg.cmd file) to overwrite installation of old driver
No error in installation process.
2. Reboot system and check all drivers and software components&APO related to Audio.
All drivers and software components&APO that related to audio are list in Device Manager with new &correct driver version, and no yellow mark.
3. Open supported audio utility(such as Dolby, Nahimic...)
Audio utility can open without error.
