Section : Preparation (10) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Tests of the X-Rite colour correction and ACM functions are all in this case,Don't execute IAA037_CF and IAA037_EF
Prepare test units and preload latest image.
The quantity of each panel source is 3.
Please execute below test steps under AC mode.
Ensure the battery level is over 20% between the test.
Before starting all the testing steps below, run both APP-A and APP-C in the APP-A folder. You can download it from the "Case tools".
Set "Dim the display" and "Put computer to sleep" to "Never" both on battery and plugged in.
The unit will not enter S3 or turn off display.
Turn off all the display power saving items, such as PSR, DPST, Vari-Bright, ALS, ELP …
The brightness will be kept.
Turn off Night light/HDR/Adaptive color.(If support)
If the machine supports dual graphics cards, please conduct all the following tests in the "Hybrid mode" .
Test images need to be displayed in full screen normally, don't adjust the zoom ratio in Photo app.
Use the "Paint" app to change the resolution of the test pictures. The resolution of the test pictures should be the same as the maximum resolution of the screen.
Color gamut and color calibration tests both test the center point of the screen.The center point of the screen can be determined by using "Center point".
All pictures required for testing must be displayed in full screen
Before testing, please leave the test unit idle for at least 30 minutes.
Before the test starts, check the sampling frequency in the APP "CA-S40" of the test device CA410. The refresh rate must be consistent with the default screen refresh rate
Path：CA-S40 APP-->Settings-->Mode-->Mode Settings-->Sync Mode-->INT
Notes: Test instrument CA410 needs to be fixed with a tripod.
