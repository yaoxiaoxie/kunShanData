4.4 Move the slider bar below to max and repeat the step above.

The “Reference Image” displays the image you have selected.
The “Image Strength” option is enabled by default and can be used to adjust similarity.
When the slider is set to maximum, the generated image should exhibit significantly higher similarity to the reference image compared to when the slider is set to 10.
If there is no noticeable difference between the results at different strength levels, please log it as a Key Issue.


About Image Strength:
At high values, image elements are more repetitive, details are clearer, and specific elements tend to stand out and attract more attention. At low values, the output image is more prompt-driven and appears more random.
4.5 Move your mouse to the image in "Reference Image", delete the image.

Prompt:
"Intelligent_test1.png" use "一个红发女孩, 坐着, 微笑"
"Intelligent_test2.png" use "一个白发老人, 站着, 蓝色大海"
"Intelligent_test3.png" use "一个金发少年, 蹲着, 森林背景".

Intelligent Control: Its main role is to guide image generation using a reference image that conveys visual style, structure, or semantic features.
This module helps maintain visual consistency between the generated image and the reference, including color tones, lighting, facial features, or general composition.
Based on this capability, Smart Reference is often used in scenarios such as multi-angle consistency, cross-image style unification, and character identity preservation.
It is especially useful in tasks like animation frame generation, concept art continuity, or generating multiple variants of a subject with stable visual features.

4.7 Upload "tile.jpg" in attachment, change the option to "Controlnet-Tile".
Use "芭蕾女舞者, 金发, 舞姿优雅" and repeat 4.3-4.4 to generate images.

Tile Control: Its main role is to add details to the photo while maintaining the overall layout of the picture.
Based on this feature, it is often combined with various plug-ins to do fine zoom processing of pictures, which is used to improve the quality of pictures with high-definition restoration.
In addition, it also has a very important function, which is used to convert the style of the picture, such as the real person manga, manga to real people, a variety of art characters production and so on.

4.8 Upload "depth.png" in attachment, change the option to "Controlnet-Depth".

Depth Control: Based on the image uploaded by the user, a depth map is extracted to perceive the spatial structure (distance and depth information) of the scene.
The model then generates an image that retains the original layout, perspective, and object positions.
Reference weight can be adjusted.

4.9 Upload "pose.jpg" in attachment, change the option to "Controlnet-Pose".

Pose Control: Based on the character picture uploaded by the user, extract the perceived skeletal pose and generate a picture with the same pose.
Reference weight can be adjusted.

Section : AI Editing (60) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Basic Edit

1. After launching the application, click the 'AI editing' module on the left side.

Successfully enter the AI editing main interface.
The sub function buttons are displayed.
(Before uploading an image, the sub function buttons are grayed out.)

2. Upload or load a sample image file.

After the image is loaded, the left side sub functions become available.
3. The functions of open, save, delete, and share on the upper right can be accessed.
Ensure that each function works properly and there are no UI abnormalities.
Crop
1. Select the crop function.
Be able to drag and resize the selection frame of the picture normally.
2. Select the ratio option in the left sidebar.
Ensure that the selection frame corresponds to the resolution and range of the selected ratio.
3. Select the size option in the left sidebar.
Ensure that the selection frame corresponds to the resolution and range of the selected size.
4. Fill in the custom resolution in the left sidebar.
Ensure that the selection frame corresponds to the correctly entered values.
5. Select the rotation option in the left sidebar.
Ensure that the picture can be rotated correctly.
6. Repeat Basic Edit Step 3.
Ensure that each function works properly and there are no UI abnormalities.
AI Cutout
1. Select the cutout function.
When no area is selected for recognition in the picture, the subject and background are automatically distinguished, and the background is selected by default in the smear mode.
2. Click 'Generate' when the mask has the background selected.
Ensure that the subject is successfully separated from the background and the selected area becomes transparent.
3. Adjust the size and draw a selection area in the smear mode.
Be able to select areas with a brush - style selection, and ensure that the brush size is adjustable.
4. Click the 'Inverse Selection' function and generate the result.
Swap the selected and unselected areas. Ensure that the generated picture is correct and the removed areas can be swapped.
5. Use the eraser function to erase the selected area.
Ensure that the selected mask can be erased.
6. Repeat Basic Edit Step 3.
Ensure that each function works properly and there are no UI abnormalities.
AI Super-Resolution
1. Upload an image with a resolution less than 512*512.
After the image is loaded, the left side sub - functions become available.
2. Select 超清 for clarity and click 'Generate'.
The image can be generated, and after saving, the resolution is indeed four times the original (matching the generated size shown on the UI).
3. Delete the image and re - upload it.
After the image is loaded, the left side sub - functions become available.

4. Select 高清 for clarity and click 'Generate'.

The image can be generated, and after saving, the resolution is indeed twice the original (matching the generated size shown on the UI).
The maximum supported resolution for generation is 2048. If exceeded, it is displayed in red and cannot be generated.
5. Repeat Basic Edit Step 3.
Ensure that each function works properly and there are no UI abnormalities.
Cloud Model Feature Group
1. Functions such as background replacement, AI image expansion, local redrawing, and smear removal are all cloud - based functions.
No need to test the functions. Just confirm that there are no obvious UI abnormalities.
