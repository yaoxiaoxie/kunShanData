Section : Preparation (20) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1.Ensure image and BIOS are appropriate.
2.Enable hibernate function. Control Panel->Hardware and Sound->Power Options->Choose what the power button does->Click Change settings that are currently unavailable->Check Hibernate->Save.
3.Modern Standby power led behavior: Please follow Modern Standby spec.
4.Music files, speech(eg: talkshow...) files and movie files
5.Keep music or speech playback with system default player(eg.media player) before check audio effects.
 
Section : Nahimic GUI check (75) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1.Check if Nahimic is list in Start Menu and Apps & Features.
Nahimic is listed in Start Menu and Apps & Features.
2.Open Nahimic and check the main window.
Nahimic UI is composed of 7 tabs (Audio, Microphone''if support'', Easy Surround, Sound Tracker, Sound Sharing, Streaming and Settings)
Microphone tab
Notes: System with Realtek Effects enabled will display this Microphone Tab on the UI(eg. Legion Go Gen2); If not, it will be hidden, Please skip Microphone Tab test.
Two microphone profiles can be chosen in Microphone Tab
Chat/Conference
Check Conference profile
Include 4 effects (Static Noise Suppression/ Echo Cancellation/ Lateral Sound Cancellation/ Voice Stabilizer)
1.Device display and volume
Microphone device and volume sync with system
2.Mute button
Microphone mute/unmute sync with system
3.ON/OFF button
Turn on/off Nahimic’s microphone effects
4.Try button: Microphone loopback on/off
Say some sentences will output user's voice
5.Close and reopen Nahimic UI
The UI still keep the last settings
6.Put the system to Sleep(MS/S3) (Please cover lid close and start menu)&Hibernate&WarmBoot&ColdBoot
The UI still keep the last settings
7.Reset button
Reset to default settings
Check Chat profile
Include 4 effects (Static Noise Suppression/ Echo Cancellation/ Lateral Sound Cancellation/ Voice Stabilizer)
1.Device display and volume
Microphone device and volume sync with system
2.Mute button
Microphone mute/unmute sync with system
3.ON/OFF button
Turn on/off Nahimic’s microphone effects
4.Try button: Microphone loopback on/off
Say some sentences will output user's voice
5.Close and reopen Nahimic UI
The UI still keep the last settings
6.Put the system to Sleep(MS/S3) (Please cover lid close and start menu)&Hibernate&WarmBoot&ColdBoot
The UI still keep the last settings
7.Reset button
Reset to default settings
Audio tab
1.1Check the default profile of Audio Tab.
The default profile:Music
1.2Check Music Profile.
Surround Sound; Volume Stabilizer; Voice; Bass; Treble;
1.3Check Movie Profile.
Surround Sound; Volume Stabilizer; Voice; Bass; Treble;
1.4Check Communication Profile.
Surround Sound; Volume Stabilizer; Voice; Bass; Treble;
1.5Check Gaming Profile.
Surround Sound; Volume Stabilizer; Voice; Bass; Treble;
2.Check Audio Device display icon.
It shows Internal Speakers.
3.Drag or click to change the volume of Speakers.
The volume can sync with system.
4.Click Mute button several times.
Mute and unmute can sync with system.
Check EQ presets and values with 'EQ presets list & values.xlsx'
5.1Check Music,Movie,Communication Gaming and Smart profile
5.2Switch profile and listen to music
Each profile can switch normally and has different effect on music.
5.3Open Equalizer window, Drag to change the 10-band Equalizer for the selected profile, listen to music.
The effect will change as you adjust
5.4Use the Reset button and the Try button to adjust your settings for the selected profile.
The effect will change as you adjust
6.1Turn off Nahimic audio effect (Click Effects On/Off button), listen to music.
All of Nahimic's audio effects would turn off and music effect will follow system default.
6.2 Click every profile.
All of Nahimic's audio effects in each profile can't be selected.
6.3Turn on Nahimic audio effect (Click Effects On/Off button).
The audio effects will be turned on and the settings are the same as before.
Surround Sound for Internal speaker.
7.1Provide 2 different modes for Internal speaker: Virtual Surround and Virtual Headset.
7.2Turn on/off Surround Sound with 2 different modes, playback surround sound audio/video files.
The effect will change.
Surround Sound for Headset. Please plug in headset for this test.
7.3You can click Personalization button under Surround Sound to configure the best match to your ears.
5 preset types are available for selection.
7.4Turn on/off Surround Sound and playback surround sound audio/video files.
The effect will change.
8.1Turn on Volume Stabilizer and listen to music.
Ensure Volume Stabilizer takes effect(the audio maintains a constant volume and it won’t be too inaudible or too loud).
8.2Click Night On/Off option.
The effect will change.
9.Drag the Voices slider between +12 and -12 DB.
It will boosts/removes the speech in movies, video games and incoming communication.
10.Drag the Bass slider between +12 and -12 DB.
It will increases/decreases the energy in low frequencies.
11.Drag the Treble slider between +12 and -12 DB.
It will increases/decreases the energy in high frequencies.
12.Click Try button.
Launches an audio sample that allows to test audio settings.
13.Close and re-open the Nahimic UI.
The UI will keep the previous settings.
14.Repeat step7-step13 under each profile.
15. Put the system to Sleep(MS/S3) (Please cover lid close and start menu)&Hibernate&WarmBoot&ColdBoot and resume to play music and open Nahimic UI window.
The effect can keep and it is the profile that select before Sleep(MS/S3) &Hibernation&WarmBoot&ColdBoot and the change of every profile can keep.
16.Click Reset button of each profile.
The profile restores the current profile to its default settings.
17.1 Plug in headphone.
Audio Device display icon will change to Headphone icon.
17.2 Click every profile and listen to music.
The effect can switch.
17.3 Plug out headphone.
The headphone icon will change back to Speaker icon.
Settings tab
1.Check Language Installed drop-down box.
The language of Nahimic's UI is based on the OS language. The languages installed on the machine in Region&language options will appear in the list and can change the language.
2.Check Version and click Website link.
Shows current version and link can get to the offical website.
3.Click Social media (only supported versions).
Get to nahimic social media pages.
4.Click Remove button(connect/config Bluetooth headset first).
Digital devices currently set as default will be restored to their original configuration.
5.Check Licence Agreement and click the link.
Link can get to the end user licence agreement website.
6.Click enabled/disabled button under Sound Tracker Engine
Make sure Sound Tracker Engine can be enabled/disabled
7.Click the Support link.
Link can get to the support website.
