Section : Preparation (440) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
BIOS/EC/Image
Flash the latest BIOS/EC and preload latest image
System Login

Login to a system using Microsoft account. Update system time and connect to WiFi AP. All the below test case please keep internet connect.
APP Installed

Note:


1. This installation step is required before each user scenario case. You can skip this step in other cases if all apps have installed in this case.


2. Plug in AC as the first section need battery capacity >80% and start to install all below APPs with latest version before scenario testing: EN OS: Chrome. Firefox, MSFT Office, Wechat, Teams, Photoshop, Tiktok, Premiere Pro, OneDrive, Adobe Acrobat Reader,VLC Mediaplay, Steam, Wegame(LOL) CH OS: MSFT Edge, Lenovo Browser,WPS, Wechat,Teams, Photoshop, QQ, Douyin, IQiyi, Tencent Meeting, Jianying, Steam, NetEase Cloud Music, Youdao translate, Baidu Netdisk, 360 Total Security, WinRAR, Sogou Pinyin(v83e as attachment), Adobe Acrobat Reader, Wegame(LOL) Both EN &CN OS: For DCC project: Download AutoCAD


3. For sogou pinyin please use the version in the attachment, this is to cover to sogou pinyin update may cause system lag issue

All APP installed without any problems and no lag obeserved
Log Clear
a. Clear the system and application event log from below path: Win+X-> Event Viewer->Windows Log->System log/Application log->Clear log in the actions window b. Clear the system reliability history: Search "View reliability history" from the windows menu, it will show all the computer's reliability and problem history, click " view all problems reports" in the bottom button, then click "clear all problem reports".
Log Tool

1. During the test record Hwinfo, ML_Scenario log.


    a. Open a command prompt window with elevated permission (Run as administrator) and navigate to the directory that contains the ML_Scenario tool, run the "ML_Scenario.bat" to start the latency log capture, and keep it in the background during the test.


    b. Open the HWinfo and start to record the sensor log



2. During the test please use camera record the lag issues and APP launch slowly issue, you can use the Potplayer to calculate the launch or response time
I/O Device
Please cover different external devices that this case will required 1. External Keyboard and Mouse include USB, WiFi and BT types 2. External Monitor 3. External Headset 4. Active Pen for Touchpanel SKU
Lag Criteria

Input lag(tying/clicking)<0.3s


UI lag(interacting with buttons, elements, menus on a UI or scrolling, animation lag)<0.5s


Light APP load<3s


Heavy APP load<5s


Gaming lag>60FPS


Video playback >25FPS
 
Section : DC 50-80% (560) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Preparation
Before start the test, make sure the battery capacity is in this range [50,80] and remove the adapter from the system. Pair the system with a BT headset and mouse
APP launch
Launch the 360 Total security, Adobe Acrobat Reader, Douyin, Sogou Pinyin, iQiyi if installed in the background, not launch the stream, photoshop, jianying, Lenovo browser if installed
Browser
Press Winkey, typing the browser name(For example: Edge) to open the web browser Edge for CN preload(Chrome if EN preload), open 10 webpage include webdownload, online video like JD/Taobao/iQiyi/Xiaohongshu/Weibo/PC.qq.com etc. Start to surf the Internet, try the web download, use mouse or Touchpad clicking, scrolling, drag the edge window. Note: don't test online video/muisc in parallel, pause one and start anoter webpage. Close the browser and repeat the browser test for 5 times
Make sure the system no lag, check ponit as below 1. Press the Winkey and the system response no lag which should less than 0.5s 2. Typing the words, the system should response in 0.3s 3. Make sure the edge/Chrome browser launch in 3s 4. When use the keyboard to input the webpage address the typing response time is lower than 300ms 5. When surfing on the webpage, clicking hyberlink/scrolling updown, the system response time is lower than 500ms.
Teams
Pause the online video/muisc play back, lauch the Teams APP 1. Connect others via message by typing seveal words 2. Drag file transfer 3. Join in a meeting con-call 4. Open a PPT and start simulate share operation link slide show, next/previous page 5. Open file manager Exit teams and repeat the teams test for 5 times
Make sure the system no lag, check ponit as below 1. Make sure the teams first UI come out less than 3s when click desktop shortcut or start from windows menu or other shorcuts 2. When use the keyboard to typing message, make sure response time is lower than 300ms 3. No audio lag or response lag when using the teams the sharing slide 5. File manager opened no delay, should less than 3s
Office
Test with Office on EN preload and WPS on CN preload. 1. Open the "Excel" APP and create a blank file 2. Try to build a 20x10 table and fill in with numbers 3. Insert a bar chart according to the data input in the last step 4. Open 5 Excel files in parallel then try some edit operations 5. Press the Win+Shift+S to caputre part of the screen than insert this capture to the excel Repeat the Excel test for 5 times
Make sure the system no lag, check ponit as below 1. Record the "Excel" APP's initial open time, make sure no more than 3s 2/3/4. Make sure that the system has no lag when typing words, creating bar charts, and the touch screen operation
Video
Connect a 4K (120Hz or above) monitor, and set display mode to extend monitor only.Check external screen setting is 4k 120hz, playing the iQiyi video æµæµªåœ°çƒ2 4K 60fps video 10 minutes. Test the youtube online video for EN preload.
Check point: 1. iQyi APP lauch no more than 3s 2. Check the FPS log that the FPS should higher than 25 FPS
Office-PPT
EN OS: 1. Open file manage and copy >20G files from one partition to another partition 2. Open the "PowerPoint" APP and create a blank presentation 3. Try to create a 10-page slide and fill it with words, pictures, tables,videos 4. Open 5 PPT files in parallel then try some edit operations 5. Click the browser icon in the taskbar and open several webpages 6. Use Win+Tab to switch from PPT window to others like Teams/Excel/File manager CH OS: Test with WPS PPT instead Repeat the PPT test for 5 times
Make sure the system no lag, check ponit as below 1. File manager should be response in 3s and no lag during the file copy 2. Record the "PowerPonit" APP initial open time, make sure no more than 3s 3. Make sure that the system has no lag when doing operations with PPT and no other flicker/no response is observed 4. No problems and lag when switching to browser too open webpages 5. When use the Win+Tab to switch, the system should be response in 0.5s
APP launch
Launch serveral APPs installed in the image like baidu netdisk, netease cloud muisc, QQ, douyin etc. Switch among the APPs by click the taskbar icon.
Check point: 1. All the APP lauch should less than 3s 2. APP switch should be smooth which shoud response in 0.5s

Stop the Hwinfo and ML_Scenario and save the two log as Project_DC80_Date. For example, S790 IML_DC80_0121.xxx
