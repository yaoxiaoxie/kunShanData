Section : Windows installation (300) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

1. Clean disk

1.1 Launch System Settings and located to Recovery -> Advanced startup, Click the button "Restart now" 

The UUT reboot and will arrived to “Choose an option” page

1.2 Click "Troubleshoot"-> "Advanced options" -> "Command Prompt" 

The CMD prompt will pop up

1.3 Input below command step by step (could refer to the attached screenshot "CleanDisk Command")：


1). diskpart


2). list disk


3). sel disk 0


4). clean


5). (If there is not only one disk displayed, repeat step 3 and 4 to clean all disks listed)


6). exit


7). click "Turn off your OC" button

The UUT clean disk successfully and shutdown

2. Manual install OS & Driver

2.1 Then Insert a boot USB disk with the OS file imported and boot from this USB disk

It will arrived to the Windows setup page successfully with no BSOD，no hang up， no error

2.2 Click Next to start the installation

OS can be normally installed with no Error，no hang up， no BSOD

2.3 OOBE and Enter system after installation finished.

2.4 Open Disk management and create about 100GB free disk space as D:\ volume

2.5 Open this PC and format D:\.

2.6 Copy related drivers to D:\ and install one by one

All drivers could be installed successfully
 
Section : Upgrade test (240) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

HDD/SDD plug in/out/upgrade test according to CPU type.If one product ship support 2 type CPU,pls pick up 2 DUT with different CPU for test

Pls cover dual/mutil storage slot config with storage sample which is included in souring plan


For example,default is single HDD config but one M.2 SSD connector free on motherboard,tester need insert ssd then upgrade to full storage configuration


For example,default is single PCIE SSD config but one sata connector free on,tester need insert HDD then upgrade to full storage configuration


If product support dual SSD Config,also cover 2nd ssd upgrade by manually
1.HDD/SDD plug in/out

1.1 Shut down,then remove both AC and battery

This section is not for EVT phase,not for all projects with onboard SSD Cover both sata and M.2 interface Cover both inhouse and ODM projects


Disable bitlocker/device encryption beforehand
1.2 Plug out HDD/SSD,then attach both AC and battery
1.3 Power on and enter BIOS menu
Ensure HDD/SSD info disappear
1.4 Shut down,then remove both AC and battery
1.5 Plug out HDD/SSD->plug in them,then attach both AC and battery
1.6 Power on and enter BIOS menu
Ensure HDD/SSD info of bios is correct
1.7 Restart and enter system
No any error during enter system
1.8 Check HDD/SSD under DM and disk management
Ensure capacity/partition/brand/model type is correct confirm system no any abnormal
1.9 Repeat steps 1.1-1.8 for 5times
Any issue happen,pls involve EE owner
2. HDD/SDD Upgrade

2.1 Shut down,then remove both AC and battery

This section is not for EVT phase,not for all projects with onboard SSD Cover both sata and M.2 interface Cover both inhouse and ODM projects


Disable bitlocker/device encryption beforehand
2.2 Replace with maximum/high configuration HDD/SSD/optane source,then attach both AC and battery
Pls refer HW OD and bios need support
2.3 Preload new system （No need preload again if system exist）and enter BIOS menu
Ensure each HDD/SSD info of information tab is correct
2.4 Restart and enter system
No any error during enter system
2.5 Check each HDD/SSD under DM and disk management
Ensure capacity/partition/brand/model type is correct confirm system no any abnormal
 
Section : Slot performance test (60) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1. Do this test section in order to measure the storage performance can meet PA SPEC or not on the Slot.
For Slot performance must follow rules as blow: 1. Slot performance must cover all slots which designed on the MB; 2.Slot performance must cover all storage sources which import to this product. If project not support slot, Please skip this section
2. Copy latest Cystaldiskmark tool to Disk.
3. Open this PC -> right click C Partion -> Select properties.
4.Select tool tab -> Click optimize button -> optimize each disk drive
5. Restart and idle at least 2 minutes.
6. Launch CDM and select slot disk as test disk.
7.Record CDM test score.
Suppose speed is not stable , you can test more times; Seq Read/Write speed should meed Lenovo PA SPEC.
