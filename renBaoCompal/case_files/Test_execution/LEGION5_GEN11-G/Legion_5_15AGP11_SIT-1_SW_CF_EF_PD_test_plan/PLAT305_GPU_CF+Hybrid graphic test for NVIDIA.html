Section : Preparation (10) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1.Refresh the newest BIOS.
2.Preload the newest OS.
3.Make sure the system only have two GPUs(one intergrated GPU and one discrete GPU).
4.Check drivers for both integrated GPU(iGPU) and discrete GPU(dGPU) are installed correctly.
 
Section : dGPU status check (140) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

1. Open NV control panel and click Desktop Tab

2. Select "Display GPU Activity Icon in Notification Area" under desktop item.

Make sure GPU Activity icon appears in the windows notification area of the taskbar
3. Go to "Manage 3D settings"-> "Global Setting"

4. Check "Preferred graphics processor" status

The default is Auto-Select

5. Click GPU Activity icon without opening any application to check the dGPU status

GPU activity is inactive and no applications run in the GPU area

6. Launch the furmark

7. Check the dGPU status with GPU activity icon

GPU activity is active and the program will run in the GPU area

8. Close the furmark and restart the system, check the dGPU status

GPU activity is inactive and no applications run in the GPU area

9. Open the QQ Music 

10. Check the dGPU status with GPU activity icon

GPU activity is inactive and QQ music program will not run in the GPU area

11. Exit the QQ Music.

12. Open NV control panel and go to "Manage 3D settings"-> "Global Setting"

13. Change ”Preferred graphics processor“ to "High-performance NVIDIA processor" and apply it

14. Restart the system

15. Open the Iqiyi app, then check the dGPU status with GPU activity icon

GPU activity is active and the Iqiyi program will run in the GPU area

16. Exit the Iqiyi app

17. Change ”Preferred graphics processor“ to "Integrated graphics" and apply it, then restart the system

18. Launch the furmark, check the dGPU status with GPU activity icon

GPU activity is active and the program will run in the GPU area

19. Close the furmark and restart the system, then check the dGPU status with GPU activity icon

GPU activity is inactive and no applications run in the GPU area
 
Section : Protected Video Playback (120) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

1.Plug into the AC adapter
2.Boot to the desktop.
3.Open NV control panel and go to "Manage 3D settings"-->"Global Setting"

4. Change "Preferred graphics processor“  to default settings and apply, then restart the system

5.Play a Blu-ray disc with the latest version of PlayerFab

Ensure Blu-ray disc can be playbacked noramlly 

6.Check the gdpu status with GPU activity icon

GPU activity is inactive and Playerfab program will not run in the GPU area.

7.Put the system into sleep(MS/S3)

Ensure system will enter to sleep successfully.
8.Plug out Power Adapter.
After plug out power adapter,system is still in MS/S3 mode.

9.Wake the system and replay the Blu-ray disc.

Blu-ray disc can continue to play without any error.

10.Check the dGPU status with GPU activity icon.

GPU activity is inactive and Playerfab program will not run in the GPU area.

11. Exit the PlayerFab app
 
Section : As pure integrated graphic test (12) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1.Boot to BIOS setup.

2.Go to Configuration-->Graphic device,change switchable graphic to Internal graphic(UMA only).Save and exit.


 


If not support UMA only mode,please skip

3.Boot to OS and open device manager-->Display adapter

Ensure there is only UMA graphic.

4. Please perform Sleep/Hibernate/Shutdown/Restart for each 5 times

system no hang up,Bsod,garbage,device lost...

5. Launch system display setting page

6. Try to change resolution/rotation/DPI

Settings can be changed successfully

7. Idle system for about one hour

System no hang up,Bsod...
