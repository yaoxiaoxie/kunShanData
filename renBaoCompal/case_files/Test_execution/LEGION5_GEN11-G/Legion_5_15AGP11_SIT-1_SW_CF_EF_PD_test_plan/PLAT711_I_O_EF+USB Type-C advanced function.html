7.Unplug AC dapter and re-plug AC adapter

There is no damage or burn out, system is not charging

10. Perform Sleep/Hibernate/CB/WB and resume the system

11. Plug in both project inbox AC power adapter and Type-C AC power adapter to system over 5 minutes

12. Perform Sleep/Hibernate/CB/WB and resume the system

13. Remove both power adapters from system

Workloading : (Min) LeadingTime : (Min) Phase :

Prepare USB 3.2 and USB4 devices for the testing,



USB3.2 Gen1 5Gbps: HIKVISION X306C or Sandisk SDDDC4


USB3.2 Gen2 10Gbps: SanDisk Extreme Pro Portable SSD or Samsung Portable SSD T7


USB3.2 Gen2x2 20Gbps: SEAGATE USB3.2 type-C PSSD  / WD Black P50


USB4 40Gbps:Oricon USB4 device


Test environment required: restart every time and then idle 1-2 mins Before testing, please do below settings: Enable write caching on the device: Device Manager > Disk drives > "Target device" >Properties > Policies > Enable Removal policy→Better performance and Write-caching policy → Enable write caching on the device


Known issue：SanDisk Extreme Pro Portable SSD and WD Black P50 will lost after resume from MS 
USB3.2 port performance
1. Download the latest CrystalDiskMark test tool

2. Connect USB3.2 storages to Type-C port.(USB3.2 Gen2*2 need test USB 3.2 Gen2*2 ports）

3. Launch the tool with default setting

For USB3.2 Gen1 5Gbps, Seq write or read speed should be over 300MB/S



For USB3.2 Gen2 10Gbps, Seq write or read speed should be over 900MB/S



For USB3.2 Gen2x2 20Gbps , Seq write or read speed should be over 2000MB/S
USB4 port performance
1. Download the latest CrystalDiskMark test tool
2. Connect USB4 storages to USB4 port

3. Launch the tool with default setting

 


For USB4 20Gbps, Seq write or read speed should be over 1700MB/S
For USB4 40Gbps , Seq write or read speed should be over 2500MB/S
4. Repeat with a TBT3 storage (sansumg X5.etc)
Seq read speed should be over 2500MB/s, Seq write speed should be over 1200MB/S

Section : Power Loading test (180) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
Graphic performance test
Note: 1. For high power consumption platform (etc. Gaming Y series), graphic P states may be set to high level in battery mode. please compare in DC mode for this platform projects 2. Please run all test in high performance mode

1. Install latest version 3DMark Timespy, run default setting for three times to get a stable score in AC mode

3Dmark runs with no error, hang up or BSOD Write down the result as critiarion score
2. For project with only one type C port, plug in a Type-C device with 10W power
System works with no error, hang up or BSOD

3. Make sure system battery capacity is over 80%, run 3Dmark in battery mode, write down the score.

The 3DMark score with 10W loading should no less than 80% of critiarion
4. For project with two type C ports, plug in 10W + 5W two Type-C devices
System works with no error, hang up or BSOD

5. Make sure system battery capacity is over 80%, run 3Dmark in battery mode, write down the score

The 3DMark score with 10W + 5W loading should no less than 75% of critiarion
Type-C function (Battery over 80%)
1. Boot system, use tool to keep CPU/GPU loading at 70% (TAT for Intel CPU and AMD tool for AMD CPU)
System work normally with no error, hang up or BSOD
2. Plug a USB device with 10W power in type A port.
System work normally with no error, hang up or BSOD
3. In battery mode, plug in a Type-C SSD
The Type-C SSD can be detected with no error
4. Copy and write 1G file between local disk and Type-C SSD
File can transfer with no error
