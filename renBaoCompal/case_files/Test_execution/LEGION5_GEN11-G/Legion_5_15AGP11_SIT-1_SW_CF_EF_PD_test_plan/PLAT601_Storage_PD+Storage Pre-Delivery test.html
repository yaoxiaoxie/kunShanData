Section : Performance Falloff test (360) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1.Large writing and limited free space

1.1 Copy latest Cystaldiskmark tool to disk

This section is only for SSD(both sata ssd and NVME PCIE ssd) and cover each disk for dual SSD config


As for CER feature PC and Y series whole , performance falloff section belong to "must test" (In case found related issue ,recommend to fix it and gating if no solution before SIT exit )


AS for CER Non-feature PC,this section belong to "optional test" (In case found related issue ,recommend to try your best to optimze SSD thermal design/FW and no need gating if no solution before SIT exit )
1.2 Open this PC->right click C partion->select properties
1.3 Select tool tab->click optimize button->optimize each disk drive
1.4 Restart and idle at least 2minutes
1.5 Launch CDM and record seq read/write speed
CDM keep default settings Suppose speed is not stable .you can test more times Seq read/write speed should meet lenovo PA spec

1.6 Install mansize 3D game and ensure play smoothly

Game test only for level3 NOTEBOOK . Anything details,pls refer attachment “NPI Performance & Battery Life Test Plan“ at PFA000_03 Performance Test Matrix
1.7 Install office and ensure deal with office document smoothly
1.8 Ensure playback online video smoothly
1.9 Copy a lot of files to disk until the biggest partition remain 40% free space
Files may contain photo,audio,video,document,game installation package
1.10 Repeat steps from 1.2-1.5
Ensure Seq read/write speed is less than 15% gap Suppose certain SSD speed is too slow ,pls confirm with storage engineer whether limitation design or not
1.11 Repeat steps from 1.6-1.8
Ensure each operation smoothly System no any abnormal No need to install game and office again
1.12 Continue to copy until the biggest partition remain around 10% free space
Once Partition from blue to red ,tester stop copy

1.13 Repeat steps from 1.2-1.5

Ensure Seq read speed is less than 15% gap,no need check seq write


Ensure copy small data no abnormal


Suppose certain SSD speed is too slow ,pls confirm with storage engineer whether limitation design or not
1.14 Repeat steps from 1.6-1.8
Ensure each operation smoothly System no any abnormal No need to install game and office again
2. Thermal throllting

2.1 Make sure Ambient Temp not higher than performance case presettings

Most ssd will trigger clock/performance drop once nand temp up to Thermal throllting, So below steps is only for SSD with default preload image


Pls cover each SSD for dual SSD config


If burnin can not show correct disk temp,you also may detect temp via other tools or method if any
2.2 Optimize each disk drive ,then restart and idle at least 2min
2.3 Launch CDM and record seq read/write speed
CDM keep default settings Suppose speed is not stable .you can test more times Seq read/write speed should meet lenovo PA spec
2.4 Install latest Burnin Test tool and launch

2.5 Open menu "Configuration\Test Selection & Duty Cycles"->Only select Disk and set to 95%, uncheck others items.


Open menu "Configuration\Test preferences\Disk"->check automatically select all fixed drives at the start of test


Test preferences->temp/battery->uncheck auto select temp ,select burnin test under monitoring application and confirm SSD source checked

No need cover HDD source for SSD+HDD config


If Burnin did not show real time temp on optane SSD/raid0,pls watch temp through other way

2.6 Begin Burnin Test for a period of time and watch SSD real time TEMP from burnin temperature monioring tab(max temp)


As for value ssd,such as each 128G source/2242 form factor/UMIS ssd,burin 15min As for mainstream ssd,burin 30min

As for CER feature PC products and Y series whole ,Ensure each SSD heavy loading temp is under 80 at summer season and under 75 for other 3 seasons (the monment of burnin compeleted ,watch max temp value)；Ensure each SSD idle temp is under 45 at summer season and under 40 for other 3 seasons (the monment of burnin beginning ,watch max temp value) As for non-feature PC ,no need to check idle&heavy loading temp except seq read/write gap not meet PA spec(You can refer ssd vendor TT temp definition if any)


If CER feature PC product SSD heavy loading temp over PA spec,but seq read/write gap can meet PA spec and thermal design can never be improved any more by confirmed with thermal RD,pls waive this defect and no need to gating Not any abnormal during burnin

2.7 After stop burnin for 2 min, Run CDM and record seq read/write speed.

Ensure Seq read/write speed is less than 20% gap Suppose certain SSD speed is too slow ,pls confirm with storage engineer whether limitation design or not
3. Low battery
3.1 Repeat steps from 1.1-1.5
CDM keep default settings Suppose speed is not stable .you can test more times Seq read/write speed should meet lenovo PA spec
3.2 Remove AC and discharge battery until below 10%
This time win10 will prompt something like "low power" from battery icon system tray
3.3 Insert AC at once ,then Launch CDM and record seq read/write speed
Ensure Seq read/write speed is less than 10% gap Suppose certain SSD speed is too slow ,pls confirm with storage engineer whether limitation design or not
 
Section : Unboxing evaluation (180) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1. Detail info

1.1 Reset PC or new preload system

This section is only for SIT and SVT phase


For CER feature PC and Y series whole ,this section is essential while for non-feature PC,it's optional


Pls test with AC mode and cover each storage source


As for dual SSD config ,only test on main DISK+C partition


If a few 3rd tool can not normally work at raid0 SSD ,pls skip related test procedure
1.2 Download latest CrystaldiskInfo and launch it
You also can use SSD-Z or AIDA64 to replace CrystaldiskInfo Ensure detail info of storage show correct,such as:type/size/brand/model name/firmware/interface/temp/controller type/form factor/NAND type If some key info show incorrect ,pls confirm with storage vendor whether real issue or not
2. Performance
2.2 Download latest Master Lu and install it
In general,Master lu disk score depand on SSD and CPU，so tester need sorting score by same CPU+diifferent SSD sku config
2.3 Launch it->select Performance test tab->select disk performance and uncheck other keypart
2.4 Click start evaluate button
Ensure disk performance score for each storage source is logical--highest source get highest score while lowest get lowest score Ensure disk score rank is similar with PCMARK10 storage score As for same CPU&same size SSD with different vendor,PA accept 10% score tolerance For example, PCMARK10 RANK---SKU5>SKU4 Expected DISK score:SKU5>SKU4 In case SKU5<SKU4 and gap =8% ,margin PASS
2.5 Download latest AS SSD benchmark,then launch it
In general,AS SSD disk score depand on SSD and CPU，so tester need sorting score by same CPU+diifferent SSD sku config
2.6 Keep default settings->click start button->record whole score of target disk
Ensure disk performance score for each storage source is logical--highest source get highest score while lowest get lowest score Ensure disk score rank is similar with PCMARK10 storage score As for same CPU&same size SSD with different vendor,PA accept 10% score tolerance
3. Stability
3.1 Download latest AIDA64 tool and install
3.2 Launch it->tools ->System stability test->select stress local disks and uncheck others
3.3 Click preferences General tab:check"Prevent the computer from going to sleep" Temperatures tab:Only select storage
3.4 Click start button to begin stability test for at least 30min
Ensure no any error/hangup occur and system work fine once stability test completed Ensure storage temp no overheat warning
