Section : Power switch test (180) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

Please check below items in better battery, better performance , Best performance by power slider


For Win11 OS:please check below items in :Balanced, Best power efficiency, Best performance by power mode
1 Power switch test in normal usage process.
1.1 Plug out AC and discharge battery to 70%;
1.2 Run system heavy loading(3DMark +HDD COPY(Tool:copy and compare) + USB mouse + external monitor)
1.3 Plug in/out AC and Repeatedly 10 times. Please plug in AC after AC icon disappeared in system tray or plug out AC after AC icon appeared in system tray.
Make sure the system don't shutdown when plug in/out AC. No abnormal happens,such as blue screen,hang up and so on.Charge battery LED work normal
1.4 Plug out battery and Repeatedly 10 times. Please plug in battery after battery icon disappeared in system tray or plug out battery after battery icon appeared in system tray. *please skip it,if battery is embedded Battery
Make sure the system don't shutdown when plug in/out battery. No abnormal happens,such as blue screen,hang up and so on. Charge battery LED work normal
1.5 Check basic function of system after stopping system heavy loading.
Make sure system work normally,work smoothly as normal Basic function are workable(Camera,Keyboard,Video,Audio,HDD copy,Touchpanel,Touchpad,Sensors,Network,I/O ports...)
1.6 Plug out battery and reinsert a battery which is less than 30% capacity,repeat above steps. Embedded battery can discharge to less than 30% capacity and repeat above steps
2 Power switch test when enable "Conservation mode" function
2.1 Keep battery and AC connected.
Make sure the battery is fully charged (100%).
2.2 Enable lenovo vantage/PC manager â€œConservation modeâ€ function, let battery discharging with AC in.
2.3 Run system heavy loading(For example:3DMark, TAT, Videoâ€¦)
2.4 Discharge battery to 70% .
2.5 Plug in/out AC and Repeatedly 10 times. Please plug in AC after AC icon disappeared in system tray or plug out AC after AC icon appeared in system tray.
Make sure the system don't enter shutdown when plug in/out AC. No abnormal happens,such as blue screen,hang up and so on. Charge battery LED work normal

2.6 Plug out battery and Repeatedly 10 times.


Please plug in battery after battery icon disappeared in system tray or plug out battery after battery icon appeared in system tray.


*please skip it,if battery is embedded Battery

Make sure the system don't shutdown when plug in/out battery. No abnormal happens,such as blue screen,hang up and so on. Charge battery LED work normal

2.8 Run step2.5 and step2.6.
