Section : Preparation (30) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1. Please refresh the latest BIOS.
2. Please make sure the test unit has refreshed correct MT Code and Family Name.

3. Check Lenovo Fn and function keys version.


Reference Path: Device Manager-> System devices-> right click Lenovo Fn and Function keys driver-> Properties-> check the Driver Version on Driver page.

If the driver is 1.X.X.X, it is Lenovo Fn and Function keys 1.0;


If the driver is 2.X.X.X, it is Lenovo Fn and Function keys 2.0.


Note: V1.0.10.3 to V1.0.10.12 are the test versions for Fn 2.0, please regard them as Fn 2.0 to test.
4. For Lenovo Fn and function keys 1.0: Open Task Manager and check whether there is Lenovo Hotkeys process, and record the time when the Lenovo Hotkeys process appears.
Lenovo Hotkeys process should start up within 30 minutes of the end of OOBE.

5.The testcase need cover EN and SC with Lenovo OS
 
Section : TOP Row Hotkey (150) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

Note1: For TB plus G5 Station ,click to switch modes, and press and hold for 2S to trigger the Lenovo smart key or Xiaotian app


Note2：The display switch for Hotkey UWP CapsLk/NumLock OSD was added from V4.6.12.0 with hotkey driver 2.0.12.1718 above.

1. Service Key

1.1 Check SMB Projects

1.1.1 Service Key for SMB projects: Check PCM Service exits.

Y:No responds(PCM handle this function). N:Move on to the next step.

1.1.2 Check baiying install.

Y: Launch baiying. N:Move on to the next step.

Y: Launch QR code. N: Launch Pcsupport URL

1.2 Service Key for Consumer Projects

1.2.1 Check Region this PRC.

Y:Check PCM Install. N:Check Vantage Install.

Note: If region is PRC ,please test 1.2.2 ; If region is not PRC ,please test 1.2.3

1.2.2 Check PCM Install

Y:Launch PCM-Services. N:Launch QR code.

1.2.3 Check Vantage Install

Y:Launch Vanatge support. N:Launch service web

1.3 Service Key for Zhaoyang Projects

1.3.1 Check PCM Install

Y:Launch PCM service page . N:Move on to the next step

1.3.2 Read user Profile Language is Chinese

Y:Launch QR Code.html _Chinese . N:Launch QR Code.html _No_Chinese

2. Check hotkeys function and OSD


Note: If the keyboard is folio keyboard, please plugging in the folio keyboard, then check the function and OSD.

The effective time of each hotkey is about 1s, and the OSD display time is about 3s.

2.1 FnLock: Press Fn+FnLock and then check its function and OSD.

Pressing Fn+FnLock can turn off/on the function of Fn key, OSD response time is acceptable and it will disappear after a while.

Please be informed before testing the following hotkeys: Hotkeys have two rows of silk-printed, one row is the hotkey icon, the other row is F1-F12. When FnLock is turned on, clicking the hotkey will excutive effect of the second row of silk-printed, clicking Fn+hotkey will excutive effect of the first row of silk-printed. When FnLock is turned off, clicking the hotkey will excutive effect of the first row of silk-printed, clicking Fn+hotkey will excutive effect of the second row of silk-printed.


Note: For gaming machines, the hotkeys bound on the keyboard may be different, please refer to the UX spec.

If the test unit has FnLock Key, please do the test （2.2-2.5）with FnLock is On and Off:

2.2 Privacy mode of camera check.

2.2.1 Privacy mode of camera: Press camera On/Off hotkey and then check its function and OSD.

Pressing camera On/Off hotkey can turn on/off privacy mode of camera, OSD response time is acceptable and it will disappear after a while.

2.2.2 Privacy mode of camera: Switch On/Off the camera button to view its functions and OSD.

The camera button enables/disables the camera's private mode. The OSD response time is acceptable and it will disappear after a period of time.

2.2.3 Privacy mode of camera: Switch On/Off the privacy shutter to view its functions and OSD.

The privacy shutter enables/disables the camera's private mode.The OSD prompt is not triggered.
Note: Touchpad hotkey and Fn+M key combination can both turn on/off touchpad. Please confirm with SWPM whether your project support Fn+M key combination firstly.

2.3 Touchpad hotkey/Fn+M:Press touchpad hotkey/Fn+M and then check its function and OSD.

Pressing touchpad hotkey/Fn+M can turn on/off touchpad, OSD response time is acceptable and it will disappear after a while.

2.4 Flight mode On/Off: Press Flight mode hotkey and then check its function.

Pressing Flight mode hotkey can turn on/off flight mode, the action response time is acceptable.

2.5 Mic On/Off: Press Mic On/Off hotkey and then check its function and OSD.

Pressing Mic On/Off hotkey can turn on/off Mic, OSD response time is acceptable and it will disappear after a while.

2.5.1 If your test unit support Mute LED: press Mic On/Off hotkey to check its function.

If the Mic on, the LED will OFF, if the Mic off, the LED will on.

2.5.2 Internal Mic test: Open Voice Recorder, set Mic On and speak into your notebook, then set Mic Off and speak into your notebook again.

The recording has sound when the Mic on, but the recording has no sound when the Mic Off.

2.5.3 External Mic test: Open Voice Recorder and insert an external Mic, set Mic On and speak into your external Mic, then set Mic Off and speak into your external Mic again.

The recording has sound when the Mic on, but the recording has no sound when the Mic Off.

3. Smart Key

3.1 Press Smart Key and then check its function.

X64:


1.Press smart key will launch AI Now(ROW)XiaoTian app (PRC) if the Fn and function driver support it (2.0.14.16 and above)


2.Update smart key logic:


Smartkey3.0: Press smart key will launch Smartkey UI.


Smartkey4.0: Press smart key will launch SE widget.


PCM or vantage dashboard is launch if Smartkey UI is not launch


ARM：


Press smart key will launch XiaoTian app if the Fn and function driver support it (3.0.1.91 and above)


ROW:Vantage


PRC:PCM/SmartEngine/Baiying 


See the attachment for the rest

3.2 Double press Smart key and check its function

Ensure it will activate the voice input mode of "Xiaotian".


Note:  


1. For Lenovo Fn and function keys driver version is 2.0.18.11 or newer


2. The test unit need preinstalled "XiaoTian"app.


3. "XiaoTian" need support the voice command as input. If not support, please update "XiaoTian" to latest version.

4. Snipping tool Key: Press Snipping tool Key and then check its function.

X50 and pre X50 projects: Pressing Snipping tool Key can launch Snipping tool or open the screenshot interface, For more details please refer to the UX spec for the current project.


X60 and newer projects: Pressing Snipping tool Key will open screenshot interface.

5. CapsLk On/Off: Press CapsLk and then check its function and OSD.

Pressing CapsLk can enable/disable capital letters, OSD response time is acceptable and it will disappear after a while.


Note:


1. On Japanese systems, there will not show CapsLk OSD.


2. For the keystrokes of CapsLk, NumLock, do not show the On-Screen-Displays by default on Legion and LOQ laptops.


If system doesn’t preinstall Lenovo Hotkey UWP, EN image can turn on/off OSD in vantage.

6. NumsLk On/Off: Press NumsLk and then check its function and OSD.

Pressing NumsLk can enable/disable number keys, OSD response time is acceptable and it will disappear after a while.


Note:


For the keystrokes of CapsLk, NumLock, do not show the On-Screen-Displays by default on Legion and LOQ laptops.


If system doesn’t preinstall Lenovo Hotkey UWP, EN image can turn on/off OSD in vantage.

7. Fn+Q: Press Fn+Q and then check its function and OSD.

Pressing Fn+Q can switch ITS/Fn+Q mode, OSD response time is acceptable and it will disappear after a while.

7.1 On the test unit installed PC Manager: Open the Battery tool bar of PC Manager in the bottom toolbar. If the icon of ITS/Fn+Q Mode is displayed, check that it matches the OSD of Fn+Q.


Note: If the testing OS is newer than 20H2, please skip this step.

The icon of ITS/Fn+Q Mode is consistent with OSD of Fn+Q.
9.2 On the test unit installed Lenovo Vantage: Open Vantage->Device->My Device Settings->Power Smart Settings, if the icon of ITS/Fn+Q Mode is displayed, check that it matches the OSD of Fn+Q.
The icon of ITS/Fn+Q Mode is consistent with OSD of Fn+Q.
 
Section : F9/F11 function test (30) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :

1.Press F9 key

On Fuji projects, AMO widget exists but not display, then lauches AMO widget (with arguments: --entry containerwidget --AMO) to the top layer. If user uninstalled AMO widget and Vantage, then check if 'PCM widget/SE widget' is available or not. And launch PCM widget if it's available. And if PCM widget is not exists, then just open 'Windows Settings>System>Power & battery page.

2.Press F11 key

1.On Fuji projects:


The Hotkey If the F11 and 'Magic Move' are enabled, then Hotkey driver shall call URI activation to trigger Unison/Magic Move. If 'Magic Move' is not installed, then the URI registration will not be there and nothing will happen. should launch the 'Magic Move' app with the URI: intel unison.


2.On non-fuji projects:


X86:The 'Smart Connect' app can be launched to the top. If the Smart Connect app is not installed on the target device, then follows above diagram to check Intel Unison app or Phone Link app.


ARM:


Lenovo Fn and Function keys driver version earlier than 3.0.3.x: launch Phonelink.


Lenovo Fn and Function keys driver version is 3.0.3.x or newer: launch "Smart Connect"(when "Smart Connect" preinstalls preload); if preload without SC, will launch PhoneLink UWP.
 
Section : Fn Function Key (60) Attachment :
Workloading : (Min) LeadingTime : (Min) Phase :
1. Fn+R
Note: The function of Fn+R 1.0 to change the refresh rate of the screen only supports some machines, please first confirm with SWPM whether the current project supports this function.

Fn+R 1.0: Open Settings-> System -> Display -> Advanced display settings to see the current refresh rate. Press Fn+R and then check its function and OSD.

After Fn+R is pressed, the refresh rate of OSD is displayed and the refresh rate of the system is switched.


The OSD is consistent with the refresh rate displayed in the system.


For Yogabook 9: If the two display panels work in different frequency,switch thw panel refresh rate according to the refresh rate of B-Display
2. Fn+Space

Note: For non-gaming machines, the Keyboard Backlight only support some projects, please confirm with SWPM whether the current project supports Smart Keyboard Backlight, Normal Keyboard Backlight or not.

2.1 Fn+Space: Press Fn+Space and check its function and OSD.

Pressing Fn+Space can switch Keyboard Backlight, OSD response time is acceptable and it will disappear after a while.


Normal Keyboard Backlight: Weak light -> High light -> Off


Smart Keyboard Backlight: Weak light -> High light -> Off->Auto light


Note: If project has different definition，please follow UX SPEC or project define

Note:


For gaming machines(X60 and X70), if the KB is non-RGB KB, please do the test 2.1.


If the KB is RGB KB, please confirm with SWPM or refer to the UX spec whether the project supports gaming OSD, if so, please check your profiles number firstly, then do the test 2..2 to 2.5, if not, skip these steps.

2.2 Fn+Space: Press Fn+Space and check its function and OSD.

Pressing Fn+Space can switch Keyboard Backlight, OSD response time is acceptable and it will disappear after a while.


5 profiles of RGB KB LED effects: Wave-> Breath-> Smooth-> Always-> Off


6 profiles of RGB KB LED effects: Screw the rainbow-> Cool blue-> Ripple-> Raining-> Audio wave left to right-> Audio wave middle to side


24-zone RGB KB LED effects: Wave->Breath->Audio ripple->Horse race lamp->Always(cool blue)->Off


Onezone RGB KB LED effects: Breath->Always (Cool blue) ->Off


Note: If project has different definition，please follow UX SPEC or project define
2.3 Fn+↑/↓: Press Fn+↑/↓ and check its function and OSD.
Pressing Fn+↑/↓ can change the global backlight brightness, OSD response time is acceptable and it will disappear after a while.
2.4 Fn+U: Press Fn+U and check its function and OSD.
Pressing Fn+U can turn on/off IO icon light, OSD response time is acceptable and it will disappear after a while.
2.5 Fn+L: Press Fn+L and check its function and OSD.
Pressing Fn+L can turn on/off Y logo light, OSD response time is acceptable and it will disappear after a while.

PS：FN+N Indicates the available version： Hotkey：2.0.2.19 + Addin 1.0.1.30XX
