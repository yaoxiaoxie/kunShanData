New/Edit test case
Test case item Expected result Test Log Mandatory
Section : Preparation Attachment :
Workloading : 5(Min) LeadingTime : 5(Min) Phase : Test Log Mandatory : N
1. Ensure image and BIOS are appropriate.
N
2. Enable hibernate function. Control Panel->Hardware and Sound->Power Options->Choose what the power button does->Click Change settings that are currently unavailable->Check Hibernate->Save.
N
3. Dolby Settings is preinstalled on PC
N
4. Modern Standby power led behavior: Please follow Modern Standby spec.
N
5. Music files and speech(eg: talkshow...) files. Notes: For some audio effect, it maybe not obvious on music file, please playback speech file to check.
N
6. Music Playback behavior under MS&S3(OS before Win11 24H2)
For MS: If playback music with Groove Music, enter MS, music playback will continue. If playback music with Windows Media Player, enter MS, music playback will pause. For S3: Playback music with any music players, enter S3, music playback will pause or stop.
N
7. Music Playback behavior under MS(For OS Win11 24H2 and higher)
playback music with media player and enter MS through a non-idle method(eg.Lid close/power button press), music will pause. playback music with windows media player legacy and enter MS through a non-idle method(eg.Lid close/power button press), music will pause or stop.
N
8. If support “AI智能场景/Smart scene”，please turn off this function to avoid automatically changing audio settings.
N
Section : Dolby Settings GUI check Attachment :
Workloading : 70(Min) LeadingTime : 70(Min) Phase : Test Log Mandatory : N
For some projects, there is Dolby logo/name on C cover of DUT, please ensure Dolby logo/name is correct with audio engineer.
N
Keep music or speech playback with system default player(eg.media player) before check audio effects.
N
1. Check if Dolby Settings is list in Start Menu and Settings->installed Apps
Dolby Settings is listed.
N
2. Open Dolby Settings-> Dolby Atmos
N
3. Check the default status
The default profile is Music，"Dolby Atmos effects for speakers and headphones" is on in more settings.
N
4. The Dynamic, Game, Movie, Music, Voice and 3 Custom profiles are shown in the page (Each profile represents a Dolby Settings profile), switch them and listen to music.
The status of profile and function description are shown in Dynamic, Game, Movie, Music, Voice profile, each profile has different effect on music and can switch normally.
N
4.1 Check Game/Movie/Music profile.
Intelligent Equzlizer(4 settings: Detailed/Balanced/Warm/Off) is in these three profiles. The default setting is Off.
N
4.2 Click Detailed/Balanced/Warm/Off buttons in each Profile and listen to music. Notes: For some music files, tester maybe can't heard the audio effect change when select Detailed/Balanced/Warm/Off, please playback pink noise file to check.
The effect will change.
N
4.5 Click default button in Game/Movie/Music Profile.
It will restore to default setting of each profile.
N
5. Select Custom1 profile
Equalizer is shown with audio spectral curve (flat by default all 0dB) in each sub-profile.
N
5.1 Change the level of frequency band and listen to music.
The effect will change.
N
5.2 Click default button in Custom1 profile
The spectral curve in Equalizer will reset to default.
N
5.3 Change the settings of Surround Virtualizer / Volume Leveler button and listen to music.
The effect will change.
N
5.4 Make some changes and select Custom 2&3 to make some different changes.
N
5.5 Select every Custom sub-profiles(Custom 1&2&3), listen to music.
The changes of every sub-profile will keep and music effect will change.
N
6.1 Select a profile and click Close button(Top right corner) to close Dolby Settings window, listen to music
Dolby Settings will disappear at Taskbar, the effect will keep.
N
6.2 Reopen Dolby Settings window, check the profile.
It is the profile that selected before close Dolby Settings window.
N
6.3 Turn off "Dolby Atmos effects for speakers and headphones" in more settings, listen to music.
Dolby Atmos effect will be off and music effect will follow system default.
N
6.4 Click every profile.
No profile can be selected.
N
6.5 Turn on "Dolby Atmos effects for speakers and headphones" in more settings, check the profile.
It is the profile that selected before turn off Dolby Settings.
N
6.6 Click Minimize button.
Dolby Settings window will minimize.
N
6.7 Restore to Dolby Settings window and click Maximize button.
Dolby Settings window will maximize and every profile can be selected.
N
6.8 Click Restore Down button nearby Minimize button.
Dolby Settings window will restore to default window.
N
7. Put the system to Sleep(MS/S3) (Please cover lid close and start menu)&Hibernate&WarmBoot&ColdBoot and resume to play music and open Dolby Settings window.
The effect can keep and it is the profile that select before Sleep(MS/S3) &Hibernation&WarmBoot&ColdBoot and the change of every profile can keep.
N
8. Click ⓘ button at top right corner to open info window
Info window will be opened, this window displays version and trademark
N
8.1 Click Privacy Policy link
Link can get to the Privacy Policy website
N
8.2 Click License link
Link can get to the License website
N
9. Check the status bar
The status bar(at the bottom right corner) displays the OEM logo and a device mode icon.
N
9.1 For DUT that support device mode, change DUT to different device mode (Notebook, Tent, Stand, Tablet mode)
Device orientation tuning icon is shown correct in Dolby Settings.
N
9.2 For DUT that not support device mode
Speaker icon will show in Dolby Settings.
N
9.3 Plug in headphone
Device mode icon will change to Headphone icon
N
For some projects, there’s no headphone models integrated so you could just see “generic/Default” now.
N
For some types of headphones, system does not recognize, the headphone selection area only displays Generic and Default.
N
9.4 Click more settings
My headphones option will be displayed，you can select headphone brand and headphone model in this option
N
9.5 Click every profile and listen to music.
The effect can switch.
N
9.6 Plug out headphone.
The headphone icon will change back to Device orientation tuning icon or Speaker icon.
N
9.7 Check the priority of Headphone/Device orientation tuning/Speaker icons.
The priority: Headphone > Device orientation tuning > Speaker A) If headphone is plugged in the device. It will display Headphone icon. B) If there is no headphone,but device supports the device mode. It will display Device orientation tuning icon. C) If there is no headphone and device doesn’t support the device mode. It will display speaker icon.
N
10. (For win10 test)Check Dolby Settings UI in High Contrast mode. Right click on Desktop and open Personalize-> High contrast settings-> High contrast-> Select every profile( #1, #2, Black, White) -> Apply and check Dolby Settings UI.
Dolby Settings UI color will follow system High contrast setting and no discordant color.
N
11. Check Dolby Settings UI in different contrast themes. Open start menu-> search and open contrast themes-> Select every theme-> Apply and check Dolby Settings UI.
Dolby Settings UI color will follow system contrast themes setting and no discordant color.
N
12. Open Settings-> Personalization-> Colors Choose your default app mode->Light or Dark option
Dolby Settings UI color will follow selected Windows theme and no discordant color.
N
AC-3 audio test (if support)
N
1. Playback AC-3 audio samples with Media player
Playback normally with Media player
N
2. Put the system to Sleep(MS/S3) (Please cover lid close and start menu)&Hibernate&WarmBoot&ColdBoot, resume and repeat step 1
Check the results
N
