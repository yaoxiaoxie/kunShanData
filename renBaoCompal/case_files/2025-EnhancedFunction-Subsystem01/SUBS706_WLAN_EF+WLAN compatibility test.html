1. Wifi Throughput compatibility test

Note1:test item 1.1 and 1.2 under test at both RSSI:-40dBm±2 and -70dBm±2（Use inssider.exe to check). For -70dBm, no need test during EVT/FVTphase. Note2:Please test without check "connect automatically" Note3:If there is no USB port, please use a type-c dongle+USB3.1 disk:SanDisk SDCZ880 USB3.1 128GB to test
N
1.1 Wifi Throughput with USB wireless Keyboard/mouse compatibility test
N
1.1.1 Let the DUT connect 2.4G network.
Ensure can connect successfully
N
1.1.2 Run cmd and ping www.baidu.com and download 3G file from the internet at the same time(when test under RSSI:-70dBm, no need download, only need ping)
Ensure can download and ping successfully
N

1.1.3 During run download, Then connect USB wireless keyboard/mouse

Ensure wifi can still connect,ping doesn't have keep time out phenomenon more than before, and the download has not distubed and the speed floating ≤30% Ensure can connect successfully
N

1.1.4 Then move USB wireless mouse and click some words by USB wireless keyborad for some seconds

Ensure wifi can still connect,ping doesn't have keep time out phenomenon more than before, and the download has not distubed and the speed the speed floating ≤30% Ensure USB wireless keyborad and mouse can used smmothly
N
1.1.5 When use USB wireless keyborad and mouse,check the status for ping and download
Ensure wifi can still connect,ping doesn't have keep time out phenomenon more than before, and the download has not distubed and the speed the speed floating ≤30%
N

1.1.6 Disconnect USB wireless keyboard/mouse

Ensure can disconnect successfully Ensure wifi can still connect,ping doesn't have keep time out phenomenon more than before, and the download has not distubed and the speed the speed floating ≤30%
N
1.1.7 Repeat 1.1.3-1.1.6 20 times
N

N
