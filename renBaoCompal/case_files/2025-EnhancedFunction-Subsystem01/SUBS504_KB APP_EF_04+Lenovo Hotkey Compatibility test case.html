Test case item Expected result Test Log Mandatory
Section : Preparation Attachment :
Workloading : 1(Min) LeadingTime : 1(Min) Phase : Test Log Mandatory : N

1. Check Lenovo Fn and function keys version.


Reference path: Open Device Manager-> System devices-> right click Lenovo Fn and Function keys driver-> Properties-> check the Driver Version on Driver page.

If the driver is 1.X.X.X, it is Lenovo Fn and Function keys 1.0;


If the driver is 2.X.X.X, it is Lenovo Fn and Function keys 2.0.


Note: V1.0.10.3 to V1.0.10.12 are the test versions for Fn 2.0, please regard them as Fn 2.0 to test.
N
2. For Lenovo Fn and function keys 1.0: Open Task Manager and check whether there is Lenovo Hotkeys process, and record the time when the Lenovo Hotkeys process appears.
Lenovo Hotkeys process should start up within 30 minutes of the end of OOBE.
N
Section : Lenovo Hotkey basic function for VOIP Key test Attachment :
Workloading : 15(Min) LeadingTime : 15(Min) Phase : Test Log Mandatory : N

Note: For Lenovo Fn and function keys 2.0: You should uninstall Lenovo Hotkeys before the following test.


Reference path: Settings -> Apps -> Apps & Features.
N

1. Press VOIP Answer Key and then check its OSD when the test unit does not install skype for business 2016 and Teams.


Note: VOIP keys only work on the Teams for work or school.

Pressing VOIP Answer Key will show the OSD to download Microsoft Teams, OSD response time is acceptable and it will disappear after a while.


Note: If the OSD shows to download Skype for Business 2016 OSD, please key a issue and wait until the issue is fixed before testing VOIP again.
N
2 Press VOIP End Key and then check its OSD when the test unit does not install Teams.
Pressing VOIP End Key will show the OSD to download Microsoft Teams, OSD response time is acceptable and it will disappear after a while.
N
3. Install Teams. Then press VOIP Answer Key to check its function when there is no Teams Audio call.
Pressing VOIP Answer Key will no response when there is no Teams Audio call.
N
4. Press VOIP Answer Key to check its function when there is a Teams Audio call coming in.
Pressing VOIP Answer Key can answer the Teams Audio call when the Teams Audio call coming in.
N
5. Press VOIP Answer Key to check its function when there is a Teams Audio call on-giong.
Pressing VOIP Answer Key will not effect the current Teams Audio call.
N
6. During a Teams Audio call on-giong, press VOIP Answer Key to check its function when a Teams Audio call coming in.
The on-giong Teams Audio call will be held, the coming in Teams Audio call will be answered.
N
7. Press VOIP End Key to check its function when there is a Teams Audio call on-giong.
Pressing VOIP End Key will not effect the current Teams Audio call.
N
8. During a Teams Audio call on-giong, press VOIP End Key to check its function when a Teams Audio call coming in.
The on going Teams Audio call will continue, the coming in Teams Audio call will be declined.
N
9. Press VOIP End Key and check its function when there is a Teams Audio call coming in.
Pressing VOIP End Key can decline the Teams Audio call.
N
10. Press VOIP End Key to check its function when there is no Teams Audio call.
Pressing VOIP End Key will no response when there is no Teams Audio call.
N
