1. Calibrate BGN Speakers 65dBA @ DUT by 'Noise_LevelCalibration.wav'.
N
2. Select Private Mode.
N
3. Start recording the audio file on the DUT and playback ‘stationary noise' file on 4 BGN Speaker.
N
4. Record the Total RMS Power and name it NS1_on.
N
5. Select Off Mode.
N
6. Repeat step3 and record the Total RMS Power of recorded file and name it NS1_off.
N
7. Compare the Total RMS Power of NS1_on&NS1_off.
NS Performance Criteria:|NS1_on - NS1_off|>25dBA
N
8. Select Shared Mode
N
9. Repeat step 3-7, record and Compare the Total RMS Power of NS2_on&NS2_off
NS Performance Criteria:|NS2_on - NS2_off|>25dBA.
N
Refer to attached picture 'Non Stationary Noise Suppression' and place Speech speaker & 4*Noise speaker.
N
Noise Type A & Type B
1. Playback 'Noise_LevelCalibration.wav' on Noise Speaker to calibrate 4 Noise speaker.
N
Ensure noise level 65dBA @ DUT position.
N
2. Playback 'Speech_LevelCalibration.wav' on Speech Speaker to calibrate Speech speaker.
N
Ensure speech level 65dBA @ DUT position.
N
3. Select Private mode.
N
4. Start recording the audio file on the DUT.
N
5. Playback 'non-stationary noise.wav' Type A noise Part from 3'30'' to 5'04'' on Noise speaker.
N
6. Record the Total RMS Power of recorded file to NS-A_on.
N
7. Select Off Mode.
N
8. Repeat step 4-5 and record the Total RMS Power of recorded file and name it NS-A_off.
N
9. Compare the Total RMS Power of NS-A_on&NS-A_off.
NS_Type A Performance Criteria:|NS-A_on - NS-A_off|>25dBA.
N
10. Select Private mode.
N
11. Start recording the audio file on the DUT.
N
12. Playback 'non-stationary noise.wav' Type B noise Part from 0'00'' to 3'21'' on Noise speaker.
N
13. Record the Total RMS Power of recorded file and name it NS-B_on.
N
14. Select Off Mode.
N
15. Repeat step 11-12 and record the Total RMS Power of recorded file and name it NS-B_off.
N
16. Compare the Total RMS Power of NS-B_on&NS-B_off.
NS_Type B Performance Criteria:|NS-B_on - NS-B_off|>25dBA.
N
17. Select Private mode.
N
18. Start recording the audio file on the DUT.
N
19. Playback 'speech play.wav' from 10'23'' to 10'44'' on Speech speaker.
N
20. Record the Total RMS Power of recorded file and name it Speech_A.
N
21. Start recording the audio file on the DUT.
N
22. Playback 'speech play.wav' from 10'56'' to 11'17'' on Speech speaker.
N
23. Record the Total RMS Power of recorded file and name it Speech_B.
N
24. Start recording the audio file on the DUT.
N
25. Playback 'speech play.wav'from 8'37'' to 10'10'' on Speech speaker.
N
26. Playback 'non-stationary noise.wav' Type A noise Part from 8'37'' to 10'10'' from Noise speaker.
N
27. Record the Total RMS Power of recorded file and name it Speech_lossA.
Please calculate the Total RMS Power of the Three Type A Noises(Speech loss) separately, make sure each noise(Speech loss) follow spec. Speech loss_Type A Criteria:|Speech_A - Speech_lossA|<=3±1dBA.
N
28. Record the audio file on the DUT.
N
29. Playback 'speech play.wav' from 5'05'' to 8'35'' on Speech speaker.
N
30. Playback 'non-stationary noise.wav' Type B noise Part from 5'05'' to 8'35'' from Noise speaker.
N
31. Record the Total RMS Power of recorded file and name it Speech_lossB.
Please calculate the Total RMS Power of the Five Type B Noises(Speech loss) separately, make sure each noise(Speech loss) follow spec. Speech loss_Type B Criteria:|Speech_B - Speech_lossB|<=3±1dBA.
N
32. Play the recorded file.
Ensure noise will be suppressed; Speech clear as expected
N
33. Playback 'Noise_LevelCalibration.wav' on Noise Speaker to calibrate Noise speaker.
N
Ensure noise level 65dBA @ DUT position.
N
34. Playback 'Speech_LevelCalibration.wav' on Speech Speaker to calibrate Speech speaker.
N
Ensure speech level 65dBA @ DUT position.(skip this step If it has been calibrated)
N
35. Select Shared mode.
N
36. Repeat step 4-9, record and compare the Total RMS Power of NS2-A_on&NS2-A_off.
NS_Type A Performance Criteria:|NS2-A_on - NS2-A_off|>25dBA.
N
37. Select Shared mode.
N
38. Repeat step 11-16, record and compare the Total RMS Power of NS2-B_on&NS2-B_off.
NS_Type B Performance Criteria:|NS2-B_on - NS2-B_off|>25dBA.
N
39. Select Shared mode.
N
40. Repeat step 18-31, record and compare the Total RMS Power of Speech2_A&Speech2_lossA
Speech loss_Type A Criteria:|Speech2_A - Speech2_lossA|<=3±1dBA.
N
41. Record and compare the Total RMS Power of Speech2_B&Speech2_lossB
Speech loss_Type B Criteria:|Speech2_B - Speech2_lossB|<=3±1dBA.
N
42. Play the recorded file.
Ensure noise will be suppressed; Speech clear as expected
