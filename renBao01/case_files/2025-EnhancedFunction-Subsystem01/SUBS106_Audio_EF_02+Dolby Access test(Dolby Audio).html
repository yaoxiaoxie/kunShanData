Test case item Expected result Test Log Mandatory
Section : Preparation Attachment :
Workloading : 5(Min) LeadingTime : 5(Min) Phase : Test Log Mandatory : N
1. Ensure image and BIOS are appropriate.
N
2. Enable hibernate function. Control Panel->Hardware and Sound->Power Options->Choose what the power button does->Click Change settings that are currently unavailable->Check Hibernate->Save.
N
3. Dolby Access is preinstalled on PC
N
4. Modern Standby power led behavior: Please follow Modern Standby spec.
N
5. Music files and speech(eg: talkshow...) files. Notes: For some audio effect, it maybe not obvious on music file, please playback speech file to check.
N
6. Music Playback behavior under MS&S3(OS before Win11 24H2)
For MS: If playback music with Groove Music, enter MS, music playback will continue. If playback music with Windows Media Player, enter MS, music playback will pause. For S3: Playback music with any music players, enter S3, music playback will pause or stop.
N
7. Music Playback behavior under MS(For OS Win11 24H2 and higher)
playback music with media player and enter MS through a non-idle method(eg.Lid close/power button press), music will pause. playback music with windows media player legacy and enter MS through a non-idle method(eg.Lid close/power button press), music will pause or stop.
N
8. If CN OS support smart engine app，please turn off ''场景调节-智能调度'' function to avoid automatically changing audio settings.
N
Section : Dolby Access GUI check Attachment :
Workloading : 70(Min) LeadingTime : 70(Min) Phase : Test Log Mandatory : N
Notes 1: For some projects, there is Dolby logo/name on C cover of DUT, please ensure Dolby logo/name is correct with audio engineer.
N
Notes 2: Keep music or speech playback with system default player(eg.media player) before check audio effects.
N

1. Check if Dolby Access is list in Start Menu and Settings->installed Apps

Dolby Access is listed.
N

For Accessibility version, Home Tab will be removed, please skip related test; Settings Tab is opened as default page.
N
2. Open Dolby Access
The HOME tab is the default landing page when you launch Dolby Access. You can find Dolby Atmos demo videos on the tab to experience the Dolby Atmos effect.
N
3. Check the default status in SETTINGS tab
The default profile is Music，"Dolby Audio effects for speakers and headphones" is on in more settings.
N
3.1 The Movie, Music, Game, Voice and Custom1 profiles are shown in the SETTINGS tab (Each profile represents a Dolby Access profile), switch them and listen to music.
The status of profile and function description are shown in Movie, Music, Game, Voice profile, each profile has different effect on music and can switch normally.
N
5. Select Custom1 profile
Equalizer is shown with audio spectral curve (flat by default all 0dB) in each sub-profile.
N
5.1 Change the level of frequency band and listen to music.
The effect will change.
N
5.2 Click default button in Custom1 profile
The spectral curve in Equalizer will reset to default.
N
6.1 Select a profile and click Close button(Top right corner) to close Dolby Access window, listen to music
Dolby Access will disappear at Taskbar, the effect will keep.
N
6.2 Reopen Dolby Access window, check the profile.
It is the profile that selected before close Dolby Access window.
N
6.3 Turn off "Dolby Audio effects for speakers and headphones" in more settings, listen to music.
Dolby Atmos effect will be off and music effect will follow system default.
N
6.4 Click every profile.
No profile can be selected.
N
6.5 Turn on "Dolby Audio effects for speakers and headphones" in more settings, check the profile.
It is the profile that selected before turn off Dolby Access.
N
6.6 Click Minimize button.
Dolby Access window will minimize.
N
6.7 Restore to Dolby Access window and click Maximize button.
Dolby Access window will maximize and every profile can be selected.
N
6.8 Click Restore Down button nearby Minimize button.
Dolby Access window will restore to default window.
N
6.9 Click Back button at the top-left corner
Dolby Access will back to the last open tab(such as home tab)
N
7. Put the system to Sleep(MS/S3) (Please cover lid close and start menu)&Hibernate&WarmBoot&ColdBoot and resume to play music and open Dolby Access window.
The effect can keep and it is the profile that select before Sleep(MS/S3) &Hibernation&WarmBoot&ColdBoot and the change of every profile can keep.
N

8. Click ⓘ button at top right corner to open info window.

Info window will be opened, this window displays version and trademark
N
8.1 Click Dolby Access Support link
Link can get to the support website
N
8.2 Click Feedback link
Link can get to the Feedback website
N
8.3 Click Rate Dolby Access link
Link can get to the Store Rating Page
N

For Accessibility version, Introduction button will be removed on info window, please skip related test.
N
8.4 Click Introduction button
Button can get to the Introduction page conisits of step-by-step tip that describe the functions of Dolby
N
8.5 Click 'What is Dolby Atoms?' link
Link can get to the Dolby Atoms website
N
8.6 Click Privacy Policy link
Link can get to the Privacy Policy website
N
8.7 Click License link
Link can get to the License website
N
9. Check the status bar
The status bar at the bottom of the SETTINGS tab displays the OEM logo and a device mode icon.
N
9.1 For DUT that support device mode, change DUT to different device mode (Notebook, Tent, Stand, Tablet mode)
Device orientation tuning icon is shown correct in Dolby Access.
N
9.2 For DUT that not support device mode
Speaker icon will show in Dolby Access.
N
Notes: For some projects, there’s no headphone models integrated so you could just see “generic/Default” now.
N
11. Plug in headphone
Device mode icon will change to Headphone icon
N
Note: For some types of headphones, system does not recognize, the headphone selection area only displays Generic and Default.
N
11.1 Click more settings in the SETTINGS tab
My headphones option will be displayed，you can select headphone brand and headphone model in this option
N
11.2 Click every profile and listen to music.
The effect can switch.
N
11.3 Plug out headphone.
The headphone icon will change back to Device orientation tuning icon or Speaker icon.
N
12. Check the priority of Headphone/Device orientation tuning/Speaker icons.
The priority: Headphone > Device orientation tuning > Speaker A) If headphone is plugged in the device. It will display Headphone icon. B) If there is no headphone,but device supports the device mode. It will display Device orientation tuning icon. C) If there is no headphone and device doesn’t support the device mode. It will display speaker icon.
N
13. (For win10)Check Dolby Access UI in High Contrast mode. Right click on Desktop and open Personalize-> High contrast settings-> High contrast-> Select every profile( #1, #2, Black, White) -> Apply and check Dolby Access UI.
Dolby Access UI color will follow system High contrast setting and no discordant color.
N
14. (For win11)Check Dolby Access UI in different contrast themes. Open start menu-> search and open contrast themes-> Select every theme-> Apply and check Dolby Access UI.
Dolby Access UI color will follow system contrast themes setting and no discordant color.
N
AC-3 audio test (if support)
N
1. Playback AC-3 audio samples with Media player
Playback normally with Media player
N
3. Put the system to Sleep(MS/S3) (Please cover lid close and start menu)&Hibernate&WarmBoot&ColdBoot, resume and repeat step 1
Check the results
N
Section : Audio Mirroring (if support) Attachment :
Workloading : 30(Min) LeadingTime : 30(Min) Phase : Test Log Mandatory : N
Bluetooth/USB/Type-c headphone no Dolby effect, only support 3.5mm headphone. Due to Dolby audio mirroring feature, Bluetooth/USB or type C headphone copy the 3.5mm headphone effect.
N
1.Connect and pair 2 Bluetooth headphones.
N
2. Open Dolby Access UI, switch to settings page
N
3. Click ''More Settings'' at the bottom-left corner of SETTINGS page
N
4. Set ''Audio Mirroring'' to ON
The primary headphone is selected automatically in the ''Source'' dropdown list
N
5. In the Target dropdown list, select the secondary headphone
The secondary headphone can be selected
N
6. Change primary and secondary headphones volume
The volume will change as you adjust
N
7. Play the music files
Ensure sound output from two headphones normally
N
8. Switch profile in settings page
Ensure sound output from two headphones normally
N
9. Trun on/off ''Dolby Audio effects for speakers and headphones'' button
The effects will change
N
10. Put the system to Sleep(MS/S3) (Please cover lid colse and start menu)and resume to play music
Ensure sound output from two headphones normally and the change can keep
N
11. Set ''Audio Mirroring'' to OFF
Primary and secondary headphones can not be selected，Ensure sound only output from primary headphone
N
12. Connect 2 USB headphones.Reselect first & second device, repeat step4-step11
N
13. Connect Bluetooth+jack headphones.Reselect first & second device, repeat step4-step11
N
14. Connect Bluetooth+USB headphones.Reselect first & second device, repeat step4-step11
N
15. Connect USB+jack headphones.Reselect first & second device, repeat step4-step11
N
16. Watch a movie with Audio Mirroring
Ensure sound output from two headphones normally
N
17. Play video games with Audio Mirroring
Ensure sound output from two headphones normally
N
