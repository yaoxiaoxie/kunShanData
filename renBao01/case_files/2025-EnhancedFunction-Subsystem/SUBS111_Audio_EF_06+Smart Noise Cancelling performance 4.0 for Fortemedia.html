Section : Beam Forming Attachment :
Workloading : 25(Min) LeadingTime : 90(Min) Phase : Test Log Mandatory : N
Refer to attached picture 'Beam Forming' and place DUT & Speech speaker
N
Private Mode
1. Calibrate Speech speaker 65dBA @DUT by 'Speech_LevelCalibration.wav'.
N
2. Select Private Mode.
N
3. Refer to attached picture '360 scale' and move DUT to a different angle, please cover (150°,180°,210°) & in-beam degree (0°,30°,330°) & out-beam degree (60°,270°,300°).
N
4. Start recording the audio file on the DUT and payback 'Speech.wav' file on Speech Speaker.
N
5. Play the recorded file.
Ensure (150°, 180°, 210°) & in-beam degree speech recording no sound glitch and discontinue.
N
6. Record the Total RMS Power of out-beam degree recorded file and fill in report.
Ensure each out-beam signal be supressed, check the out-beam difference value>20±2db.
N
7. Refer to attached picture 'Beam Forming' and place in-beam speaker(0deg)&out-beam speaker(45deg/90deg).
N
Calibrate in-beam&out-beam Speech speaker 65dBA @DUT by 'Speech.wav' .
N
8. Start recording the audio file on the DUT and playback 'Speech.wav' file on in-beam speaker.
N
9. Record the Total RMS Power and name it in-beam_0deg.
N
10. Play the recorded file.
Ensure no sound glitch and discontinue.
N
11. Record the audio file on the DUT and playback 'Speech for outbeam.wav' file on out-beam(45deg) Speaker.
N
12. Record the Total RMS Power and name it out-beam_45deg. Compare the recording RMS Power of in-beam_0deg&out-beam_45deg.
|out-beam_45deg - in-beam_0deg|>20dBA.
N
13. Record the audio file on the DUT and playback 'Speech for outbeam.wav' file on out-beam(90deg) Speaker.
N
14. Record the Total RMS Power to outbeam_90deg. Compare the recording RMS Power of in-beam_0deg&out-beam_90deg.
|out-beam_90deg - in-beam_0deg|>20dBA.
N
15. Start recording the audio file on the DUT and playback 'Speech.wav' file on in-beam speaker(0deg) & playback 'Speech for outbeam.wav' file on out-beam Speaker(45deg or 90deg) simultaneously.
N
16. Play the recorded file.
Ensure no sound glitch and discontinue, in beam speech clear as expected.

Section : Howling Check Attachment :
Workloading : 15(Min) LeadingTime : 55(Min) Phase : Test Log Mandatory : N
Notes 1:Change 'user account control settings' to 'never notify'
N
Notes 2:Turn off windows defender firewall and Antivirus Software(such as McAfee)
N
Howling check
1.Prepare two units that have passed microphone airtightnesss test.Refer to 'howling environment' and place two units
N
2.Install and login WeChat on two units，Change the MIC&Speaker volume to maximum on two units
N

If installed WeChat app is 4.0.0 or newest version, please turn on (AI 智能场景/Modes-automatic) in PCM/Vantage, make sure under Meeting mode
N
3.Select environment mode on two units and establish a voice call between two units(distance of A&B keep 40cm)
ensure howling noise is not be suppressed during voice call
N
4.Select private mode on two units，establish a voice call between two units
No howling noise exist during voice call，a little howling noise can be accepted
N
5.Install howling test tool on two units，Connect to the same AP
N
6.Open howling test tool on two units，select test unit as DUT PC，select the other as HOST PC
N
7.Input IP address displayed by host on DUT interface，Waiting for DUT to connect to HOST automatically
Howling test tool will show connected on two units
N
8.Click 'Disable sound effect on DUT' button on test tool of DUT or Host PC
Tool will automatically synchronize operation on two units
N
9.Uncheck 'Enable audio enhancements and Apply' on Microphone properties of Host PC
N
10.Establish a voice call between two units
N
11.When hear the howling. Click the 'Start' button on the tool of DUT PC or Host PC
Make sure the results are all pass
N
