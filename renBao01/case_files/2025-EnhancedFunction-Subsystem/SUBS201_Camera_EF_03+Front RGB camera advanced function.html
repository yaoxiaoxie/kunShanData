Section : Image resolution auto test Attachment :
Workloading : 25(Min) LeadingTime : 1280(Min) Phase : Test Log Mandatory : N
Important:This section of the test requires the use of a power distribution unit (PDU)!
N
The name of the tested machine cannot have Chinese characters or spaces. Will affect automated testing.
N
For the automation testing to proceed normally, please temporarily delete all password/PIN/facial recognition.
N
Uninstall antivirus software or turn off firewalls.
N
When running the following tool, please do it in the default mode of your computer. (If you have a machine that supports dual graphics cards, please make sure it is in the default mode)
N
Please turn down the screen brightness appropriately without affecting the viewing effect, such as 20%.
Other power consuming ones such as keyboard light can also be turned off
N
Open Windows camera app,Change the preview screen to "Take photo"
Ensure the camera previews properly.
N
Ensure the camera app opens on the photo interface before each run.
N
Close Windows camera app.
N
Camera_tool can be obtained from the "Case tool".
N
Double-click to open "Camera_Test_Tool.exe"
N
When the "Camera_Tool" interface appears,manually fill in the three requiredinformation for "Product/Phase/SKU".
N
Click "Test",Switch to the test configuration interface
N
Enter the "PDU IP" address and "PDU Port" number
Note:"PDU IP" address and "PDU Port" number are both the attribute information of the PDU itself. Please ask the equipment owner.
N
Important:The following testing steps,the automatic tool has two settings and testing methods. Please select the corresponding steps according to the project series: ①Camera_V1: support GenX series before the model (not including GenX series) . ②Camera_V2: support from the Genx series onwards (including GenX series).
N
Camera_V1
After completing [Enter the "PDU IP" address and "PDU Port" number].
N
Set the number of test rounds as follows: 1.Photo --> S5 --> AC Cycles, set the value to 0; 2.Photo --> S5 --> DC Cycles, set the value to 0; 3.Video --> S5 --> AC Cycles, set the value to 0; 4.Video --> S5 --> DC Cycles, set the value to 0; 5. No changes are required for any other settings.
Note:Set all the test cycles of S5 to 0. Because for machines before the GEN X series, the test tool cannot enable the machine to automatically recover from S5.
N
Click "Get resulution" and wait for the software to read the resolution automatically
N
After reading the resolution, click "OK" on the pop-up window "Get resolution Finish!".
N
Click "Start" to start testing.
N
See the pop-up box "All test item finish", the automatic test run is finished.
N
Close Camera_Tool.
N

Note:If there is a fail in the report, please check the reason by yourself, generally there is no error in the running of the programme means that there is no problem with the programme.
